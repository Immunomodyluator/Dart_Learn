# 17.2 Dart для веба

## 1. Формальное определение

Dart компилируется в JavaScript (через **dart2js**) или WebAssembly (через **dart2wasm**) для запуска в браузере. Пакет `package:web` предоставляет типобезопасные привязки к Web API (DOM, Fetch, Canvas и др.).

## 2. Зачем это нужно

- **Типобезопасность** — статическая типизация для фронтенда.
- **Один язык** — общий код между клиентом и сервером.
- **Tree shaking** — dart2js удаляет неиспользуемый код.
- **WebAssembly** — dart2wasm даёт производительность близкую к нативной.

## 3. Настройка проекта

```bash
# Создание web-проекта
dart create -t web my_web_app

# Установка webdev
dart pub global activate webdev

# Запуск dev-сервера с hot reload
webdev serve

# Сборка для production
webdev build
```

```yaml
# pubspec.yaml
dependencies:
  web: ^1.0.0

dev_dependencies:
  build_runner: ^2.4.0
  build_web_compilers: ^4.0.0
```

## 4. Работа с DOM (package:web)

```dart
// web/main.dart
import 'package:web/web.dart';

void main() {
  final app = document.querySelector('#app') as HTMLDivElement;

  // Создание элементов
  final heading = document.createElement('h1') as HTMLHeadingElement;
  heading.textContent = 'Hello from Dart!';
  app.appendChild(heading);

  // Обработка событий
  final button = document.createElement('button') as HTMLButtonElement;
  button.textContent = 'Click me';
  button.onClick.listen((event) {
    counter++;
    output.textContent = 'Clicked $counter times';
  });
  app.appendChild(button);

  var counter = 0;
  final output = document.createElement('p') as HTMLParagraphElement;
  output.textContent = 'Clicked 0 times';
  app.appendChild(output);
}
```

## 5. Fetch API

```dart
import 'dart:convert';
import 'dart:js_interop';
import 'package:web/web.dart';

Future<void> loadData() async {
  final response = await window.fetch('/api/data'.toJS).toDart;

  if (!response.ok) {
    throw Exception('HTTP ${response.status}');
  }

  final text = await response.text().toDart;
  final data = jsonDecode(text.toDart);
  print(data);
}
```

## 6. Компиляция

```bash
# Development (dart2js с проверками, source maps)
webdev serve

# Production (минифицированный JS)
webdev build --release

# Размер выходного файла
# dart2js с tree-shaking: обычно 100-500 KB (gzip)
```

### dart2wasm (WebAssembly)

```bash
# Компиляция в Wasm (экспериментально)
dart compile wasm web/main.dart -o web/main.wasm

# Требует специальный JS-загрузчик
```
