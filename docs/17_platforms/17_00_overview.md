# 17. Платформенные применения Dart — обзор

## О чём этот раздел

Dart — мультиплатформенный язык. Он используется на сервере, в вебе, для мобильных и десктопных приложений (через Flutter), а также для создания CLI-утилит. Этот раздел обзорно покрывает основные области применения.

## Ключевые понятия

```
┌───────────── Платформы Dart ──────────────────────┐
│                                                    │
│  Сервер (dart:io)                                  │
│  ├── HTTP-серверы (shelf, dart_frog)               │
│  ├── REST / GraphQL API                           │
│  └── Микросервисы, CLI-демоны                     │
│                                                    │
│  Веб (dart2js, dart2wasm)                          │
│  ├── SPA на чистом Dart                           │
│  ├── Компиляция в JS / WebAssembly                │
│  └── package:web для DOM                          │
│                                                    │
│  Flutter (мобайл, десктоп, веб)                    │
│  ├── iOS, Android                                 │
│  ├── Windows, macOS, Linux                        │
│  └── Flutter Web                                  │
│                                                    │
│  CLI                                               │
│  ├── Компиляция в нативный бинарник               │
│  ├── Аргументы, I/O, файлы                        │
│  └── Распространение через pub global             │
│                                                    │
└────────────────────────────────────────────────────┘
```
