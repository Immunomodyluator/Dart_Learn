# 3. Встроенные типы и литералы

## Обзор раздела

Dart имеет набор встроенных типов, являющихся частью `dart:core`. Все они — полноценные объекты с методами и свойствами, но при этом оптимизированы на уровне VM для максимальной производительности.

Этот раздел охватывает:

1. **Числа и операции** — `int`, `double`, арифметика, преобразования.
2. **Строки и интерполяция** — работа со строками, многострочные литералы.
3. **Булевы значения** — `bool` и логические операции.
4. **Runes и Symbols** — юникод-руны и символы.

### Связь с другими разделами

Встроенные типы — строительные блоки для коллекций (раздел 4), параметров функций (раздел 6) и полей классов (раздел 7). Null Safety (раздел 2.5) применяется ко всем этим типам.
