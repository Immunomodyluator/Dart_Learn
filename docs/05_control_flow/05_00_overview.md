# 5. Управление потоком

Управление потоком (control flow) определяет порядок выполнения инструкций в программе. Dart предоставляет классические конструкции — `if/else`, `switch`, циклы — но с важными улучшениями Dart 3: **switch expressions**, **exhaustive checking** и **pattern matching**, превращающие `switch` из примитивного оператора в мощный инструмент деструктуризации.

Все конструкции управления потоком в Dart работают **только с `bool`** — нет неявных приведений (truthy/falsy), что устраняет целый класс ошибок, типичных для JavaScript.
