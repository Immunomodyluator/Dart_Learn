# 2. Основы синтаксиса

## Обзор раздела

Dart — объектно-ориентированный язык со статической типизацией, C-подобным синтаксисом и sound null safety. Если вы знаете Java, C#, Kotlin или TypeScript — синтаксис Dart покажется знакомым, но с важными отличиями в деталях.

Этот раздел охватывает фундаментальные конструкции языка:

1. **Hello World и структура программы** — точка входа `main()`, файлы, импорты.
2. **Объявление переменных** — `var`, `dynamic`, `Object` и когда что использовать.
3. **Типы и вывод типов** — статическая типизация с инференсом.
4. **const и final** — константы времени компиляции и времени выполнения.
5. **Null Safety** — звуковая безопасность обработки null.

### Связь с другими разделами

Без понимания основ синтаксиса невозможно двигаться дальше. Переменные и типы — основа для коллекций (раздел 4), функций (раздел 6) и ООП (раздел 7). Null Safety пронизывает весь язык начиная с Dart 2.12.
