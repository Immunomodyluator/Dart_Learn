# 16. Архитектура кода и паттерны — обзор

## О чём этот раздел

Организация кода в Dart-проектах: модульная структура, паттерны проектирования, управление зависимостями, неизменяемые данные и антипаттерны.

## Ключевые понятия

```
┌─────────── Архитектура Dart-проекта ──────────────┐
│                                                    │
│  Структура                                         │
│  ├── Разделение на пакеты и модули                │
│  ├── Логические слои (data / domain / presentation)│
│  └── Barrel-файлы и экспорты                      │
│                                                    │
│  Паттерны                                          │
│  ├── Dependency Injection                         │
│  ├── Repository / Service                         │
│  ├── Value Objects                                │
│  └── Immutable data                               │
│                                                    │
│  Качество                                          │
│  ├── SOLID-принципы                               │
│  ├── Антипаттерны                                 │
│  └── Чистый код                                   │
│                                                    │
└────────────────────────────────────────────────────┘
```
