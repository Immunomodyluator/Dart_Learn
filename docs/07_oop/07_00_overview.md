# 7. Объектно-ориентированное программирование — обзор

Dart — **объектно-ориентированный** язык с единым корнем иерархии (`Object`). Всё в Dart — объект: числа, строки, функции, `null` — все наследуют от `Object?`.

## Ключевые особенности ООП в Dart

- **Единая иерархия** — все типы наследуют от `Object` (или `Object?` с null safety).
- **Классовые модификаторы** (Dart 3) — `sealed`, `final`, `base`, `interface`, `mixin` управляют наследованием за пределами библиотеки.
- **Нет множественного наследования** — вместо этого mixins.
- **Implicit interfaces** — каждый класс автоматически объявляет интерфейс.
- **Extension methods** — добавление методов к существующим типам без наследования.
- **Extension types** (Dart 3.3) — zero-cost wrapper для статической типизации.
