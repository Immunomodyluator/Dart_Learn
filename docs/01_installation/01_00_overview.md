# 1. Установка и инструменты

## Обзор раздела

Dart — компилируемый язык с собственной виртуальной машиной, SDK и экосистемой пакетов. Перед написанием первой строки кода необходимо установить инструментарий и понять, как устроен цикл разработки: от редактора кода до анализа и форматирования.

Этот раздел охватывает четыре аспекта инструментальной подготовки:

1. **SDK и редакторы** — установка Dart SDK, выбор IDE, настройка плагинов.
2. **CLI-инструменты** — команда `dart`, компиляция, анализ и запуск из терминала.
3. **Менеджер пакетов pub** — управление зависимостями через `pubspec.yaml`.
4. **Форматирование и анализ** — автоформатирование, статический анализ, правила линтинга.

### Связь с другими разделами

Знание инструментов — фундамент для всех последующих тем. Без установленного SDK невозможно запустить примеры из разделов 2–20. Понимание `pub` необходимо для работы с пакетами (раздел 12), а навыки анализа и линтинга — для CI/CD (раздел 18) и архитектуры (раздел 16).
