# 6. Функции и замыкания

Функции — основной строительный блок Dart-программ. В Dart функции являются **объектами первого класса** (first-class objects): их можно присваивать переменным, передавать как аргументы, возвращать из других функций и хранить в коллекциях.

Dart поддерживает именованные функции, анонимные (лямбды), стрелочные (arrow), а также **замыкания** — функции, захватывающие переменные из окружающей области видимости. Система параметров богата: позиционные, именованные, опциональные, required, значения по умолчанию.

Начиная с Dart 2.13+, поддержка **typedef** расширена до произвольных типов, а в Dart 3 добавлены **records**, которые часто используются вместо кортежей для возврата нескольких значений.
