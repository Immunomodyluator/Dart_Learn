# 14. Interop: FFI и Web — обзор

## О чём этот раздел

Dart может взаимодействовать с кодом на других языках: вызывать нативные библиотеки C/C++ через FFI (Foreign Function Interface) и работать с JavaScript в браузерных приложениях. Также рассматриваются ограничения рефлексии в Dart.

## Ключевые понятия

```
┌───────────────── Interop в Dart ──────────────────┐
│                                                    │
│  Dart FFI (dart:ffi)                               │
│  ├── Вызов C/C++ функций                          │
│  ├── Работа с указателями и памятью               │
│  └── Поддержка: Native (iOS, Android, Desktop)    │
│                                                    │
│  JS Interop (dart:js_interop)                     │
│  ├── Вызов JavaScript функций                     │
│  ├── Работа с DOM и Web API                       │
│  └── Поддержка: Web (dart2js, dart2wasm)          │
│                                                    │
│  Рефлексия                                         │
│  ├── dart:mirrors (только JIT, deprecated)        │
│  └── Альтернатива: кодогенерация                  │
│                                                    │
└────────────────────────────────────────────────────┘
```
