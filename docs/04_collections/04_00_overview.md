# 4. Коллекции

Коллекции — базовые структуры данных для хранения и обработки наборов значений. Dart предоставляет три основных типа коллекций: **List** (упорядоченный список), **Set** (множество уникальных элементов) и **Map** (ассоциативный массив ключ→значение).

В отличие от многих языков, Dart интегрирует коллекции в литеральный синтаксис: `[]`, `{}` и `{key: value}`. Dart 2.3+ добавил мощные возможности — **spread-оператор** (`...`) и **collection if/for** — позволяющие строить коллекции декларативно, что особенно важно в Flutter для построения UI-деревьев.

Все коллекции Dart строго типизированы (дженерики), реифицированы в runtime и поддерживают rich API через `Iterable` — `map`, `where`, `fold`, `reduce` и десятки других методов функционального стиля.
