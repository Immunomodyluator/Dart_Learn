# Задание 05 (4.3): Булевы значения — 🔲 Дополнить код

**Тема:** 04. Встроенные типы и литералы | **Подтема:** 4.3 Булевы значения

---

Заполни пропуски: запиши корректные булевы выражения для каждого условия.

```dart
void main() {
  int age = 20;
  bool hasTicket = true;
  String role = 'admin';

  // Совершеннолетний И имеет билет
  bool canEnter = ___ && ___;

  // Не является админом
  bool isNotAdmin = ___;

  // Возраст от 18 до 65 включительно
  bool workingAge = ___ && ___;

  // Имеет билет ИЛИ является админом
  bool hasAccess = ___ || ___;

  print(canEnter);    // true
  print(isNotAdmin);  // true
  print(workingAge);  // true
  print(hasAccess);   // true
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  int age = 20;
  bool hasTicket = true;
  String role = 'admin';

  bool canEnter = age >= 18 && hasTicket;
  bool isNotAdmin = role != 'admin';
  bool workingAge = age >= 18 && age <= 65;
  bool hasAccess = hasTicket || role == 'admin';

  print(canEnter);    // true
  print(isNotAdmin);  // false (role == 'admin')
  print(workingAge);  // true
  print(hasAccess);   // true
}
```

Обрати внимание: `isNotAdmin` вернёт `false`, потому что `role == 'admin'`. В Dart нет неявного приведения к `bool` — условие в `if` обязательно должно быть типа `bool`.

</details>
