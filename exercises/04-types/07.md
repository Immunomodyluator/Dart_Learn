# Задание 07 (4.4): Runes и Symbols — 🖊 Написать код

**Тема:** 04. Встроенные типы и литералы | **Подтема:** 4.4 Runes и Symbols

---

Напиши функцию `countEmoji(String text)`, которая возвращает количество emoji (символов вне Basic Multilingual Plane, т.е. code point > 0xFFFF) в строке. Подсказка: используй `text.runes`.

<details>
<summary>Решение</summary>

```dart
int countEmoji(String text) {
  return text.runes.where((r) => r > 0xFFFF).length;
}

void main() {
  print(countEmoji('Hello 🎉 World 🚀')); // 2
  print(countEmoji('No emoji here'));      // 0
  print(countEmoji('👨‍💻 Coder'));            // 1 (базовый символ; ZWJ-последовательности считаются иначе)
}
```

`String.runes` — итерируемая последовательность Unicode code points. Emoji обычно имеют code point выше `0xFFFF` (они находятся в плоскостях выше BMP). `String.length` в Dart считает UTF-16 code units, а не символы — поэтому для корректной работы с emoji нужны `runes`.

</details>
