Функция считает среднее арифметическое, но возвращает неверный результат. Найди проблему.

```dart
int average(List<int> nums) {
  return nums.reduce((a, b) => a + b) / nums.length;
}

void main() {
  print(average([10, 20, 30])); // ожидается: 20
}
```

<details>
<summary>Решение</summary>

```dart
// Вариант 1: целочисленное деление
int average(List<int> nums) {
  return nums.reduce((a, b) => a + b) ~/ nums.length;
}

// Вариант 2: вернуть double
double averageDouble(List<int> nums) {
  return nums.reduce((a, b) => a + b) / nums.length;
}

void main() {
  print(average([10, 20, 30]));       // 20
  print(averageDouble([10, 20, 30])); // 20.0
}
```

`/` в Dart всегда возвращает `double`, а тип возврата объявлен `int` — ошибка компиляции. Нужно либо использовать `~/` (целочисленное деление), либо изменить тип возврата на `double`.

</details>
