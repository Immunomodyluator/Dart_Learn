Напиши функцию `formatPrice(double price, String currency)`, которая возвращает строку вида `"Price: $12.50 USD"`. Используй строковую интерполяцию и метод форматирования числа с двумя знаками после запятой.

<details>
<summary>Решение</summary>

```dart
String formatPrice(double price, String currency) {
  return 'Price: \$${price.toStringAsFixed(2)} $currency';
}

void main() {
  print(formatPrice(12.5, 'USD'));   // Price: $12.50 USD
  print(formatPrice(1999.9, 'EUR')); // Price: $1999.90 EUR
}
```

`toStringAsFixed(2)` округляет и форматирует число с двумя десятичными знаками. В строковой интерполяции `\$` — экранированный знак доллара (литерал), `${}` — вычисляемое выражение. `$currency` — простая интерполяция переменной.

</details>
