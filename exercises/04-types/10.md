Код не компилируется. Найди все ошибки в работе с records.

```dart
void main() {
  var point = (x: 10, y: 20);

  // Попытка изменить поле record
  point.x = 30;

  // Деструктуризация
  var (a, b) = point;
  print('$a $b');

  // Сравнение
  var p2 = (x: 10, y: 20);
  print(point == p2); // ожидается: true
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  var point = (x: 10, y: 20);

  // Ошибка 1: records иммутабельны — поля нельзя изменять
  // point.x = 30; // убрать эту строку
  // Для "изменения" нужно создать новый record:
  point = (x: 30, y: point.y);

  // Ошибка 2: именованные поля деструктурируются через (:name), не позиционно
  var (:x, :y) = point;
  print('$x $y'); // 30 20

  var p2 = (x: 10, y: 20);
  // point теперь (x:30, y:20), а p2 — (x:10, y:20) → false
  var p3 = (x: 30, y: 20);
  print(point == p3); // true — records сравниваются структурно
}
```

Две ошибки: 1) Records иммутабельны — нельзя присвоить значение полю. 2) Для именованных полей деструктуризация использует `(:x, :y)`, а не `(a, b)` (позиционный синтаксис работает только для позиционных полей).

</details>
