Заполни пропуски: используй нужный оператор деления или метод преобразования.

```dart
void main() {
  // Обычное деление → double
  double a = 7 ___ 2;         // 3.5

  // Целочисленное деление (truncate) → int
  int b = 7 ___ 2;            // 3

  // Остаток от деления
  int c = 7 ___ 2;            // 1

  // Преобразовать double в int (отбросить дробь)
  int d = 3.9.___;            // 3

  // Преобразовать int в double
  double e = 5.___;           // 5.0

  print('$a $b $c $d $e');
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  double a = 7 / 2;       // 3.5
  int b = 7 ~/ 2;         // 3
  int c = 7 % 2;          // 1
  int d = 3.9.toInt();    // 3
  double e = 5.toDouble(); // 5.0

  print('$a $b $c $d $e');
}
```

`/` всегда возвращает `double`. `~/` — целочисленное деление (truncate). `%` — остаток. `.toInt()` отбрасывает дробную часть (не округляет). `.toDouble()` преобразует `int` в `double`.
