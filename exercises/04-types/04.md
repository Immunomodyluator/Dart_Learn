# Задание 04 (4.2): Строки и интерполяция — 🐛 Найти баг

**Тема:** 04. Встроенные типы и литералы | **Подтема:** 4.2 Строки и интерполяция

---

Функция должна вернуть строку `"Hello, ALICE!"`, но возвращает другое. Найди баг.

```dart
String shout(String name) {
  return 'Hello, $name.toUpperCase()!';
}

void main() {
  print(shout('alice')); // ожидается: Hello, ALICE!
}
```

<details>
<summary>Решение</summary>

```dart
String shout(String name) {
  return 'Hello, ${name.toUpperCase()}!';
}

void main() {
  print(shout('alice')); // Hello, ALICE!
}
```

`$name.toUpperCase()` в строке интерпретируется как `$name` (переменная) + `.toUpperCase()!` (литерал). Для вызова метода внутри интерполяции нужно обернуть выражение в `${}`: `${name.toUpperCase()}`.

</details>
