Дополни код: выведи строку посимвольно (по кодовым точкам Unicode), а затем преобразуй список code points обратно в строку.

```dart
void main() {
  const text = 'Hi!';

  // Получить итерацию по Unicode code points
  final runes = text.___;

  // Вывести каждый code point
  for (final r in runes) {
    print(r); // 72, 105, 33
  }

  // Собрать строку обратно из code points
  final restored = String.___(runes);
  print(restored); // Hi!
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  const text = 'Hi!';

  final runes = text.runes;

  for (final r in runes) {
    print(r); // 72, 105, 33
  }

  final restored = String.fromCharCodes(runes);
  print(restored); // Hi!
}
```

`String.runes` — геттер, возвращающий `Runes` (итерируемую последовательность `int` code points). `String.fromCharCodes(Iterable<int>)` — фабричный конструктор, который собирает строку из code points.
