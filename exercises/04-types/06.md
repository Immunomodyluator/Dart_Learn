# Задание 06 (4.3): Булевы значения — 🐛 Найти баг

**Тема:** 04. Встроенные типы и литералы | **Подтема:** 4.3 Булевы значения

---

Код не компилируется. Найди все ошибки, связанные с булевой логикой в Dart.

```dart
void main() {
  int count = 5;
  String name = '';

  if (count) {
    print('Has items');
  }

  if (!name) {
    print('Name is empty');
  }

  bool result = 1 == 1;
  print(result == true ? 'yes' : 'no');
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  int count = 5;
  String name = '';

  if (count > 0) {           // int не приводится к bool автоматически
    print('Has items');
  }

  if (name.isEmpty) {        // String не приводится к bool; используем .isEmpty
    print('Name is empty');
  }

  bool result = 1 == 1;
  print(result ? 'yes' : 'no'); // result уже bool, == true избыточно (но не ошибка)
}
```

Dart строго типизирован: `if (count)` и `if (!name)` — ошибки компиляции, т.к. `int` и `String` не являются `bool`. Нужны явные условия: `count > 0`, `name.isEmpty`, `name.isNotEmpty` и т.д.

</details>
