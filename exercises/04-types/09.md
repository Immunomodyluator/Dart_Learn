Напиши функцию `minMax(List<int> nums)`, которая возвращает **record** с двумя именованными полями: `min` и `max`. Вызови её и выведи оба значения через деструктуризацию.

<details>
<summary>Решение</summary>

```dart
({int min, int max}) minMax(List<int> nums) {
  return (min: nums.reduce((a, b) => a < b ? a : b),
          max: nums.reduce((a, b) => a > b ? a : b));
}

void main() {
  final result = minMax([3, 1, 7, 2, 9, 4]);

  // Деструктуризация через именованные поля
  final (:min, :max) = result;
  print('min=$min, max=$max'); // min=1, max=9

  // Или напрямую
  print(result.min); // 1
  print(result.max); // 9
}
```

Record — анонимный иммутабельный кортеж. Тип записывается как `({int min, int max})`. Именованные поля доступны через `.min`, `.max` или через деструктуризацию `(:min, :max)` (Dart 3 pattern matching).

</details>
