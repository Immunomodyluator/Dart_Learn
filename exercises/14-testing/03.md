Реализуй тесты с ручными mock-объектами (без внешних пакетов).

**Дано:**

```dart
abstract interface class UserRepository {
  Future<User?> findById(int id);
  Future<void> save(User user);
}

class User {
  final int id;
  final String name;
  User(this.id, this.name);
}

class UserService {
  final UserRepository repo;
  UserService(this.repo);

  Future<String> getUserName(int id) async {
    final user = await repo.findById(id);
    if (user == null) throw StateError('User $id not found');
    return user.name;
  }

  Future<User> createUser(int id, String name) async {
    final existing = await repo.findById(id);
    if (existing != null) throw StateError('User $id already exists');
    final user = User(id, name);
    await repo.save(user);
    return user;
  }
}
```

**Задача:**

1. Создай `MockUserRepository implements UserRepository` — хранит пользователей в `Map`, отслеживает вызовы `save` в `List<User> savedUsers`.
2. Напиши тесты:
   - `getUserName` возвращает имя существующего пользователя
   - `getUserName` бросает `StateError` если пользователь не найден
   - `createUser` создаёт и сохраняет пользователя
   - `createUser` бросает `StateError` если пользователь уже существует

```dart
import 'package:test/test.dart';

// --- Production code ---
abstract interface class UserRepository {
  Future<User?> findById(int id);
  Future<void> save(User user);
}

class User {
  final int id;
  final String name;
  User(this.id, this.name);
  @override
  String toString() => 'User($id, $name)';
}

class UserService {
  final UserRepository repo;
  UserService(this.repo);

  Future<String> getUserName(int id) async {
    final user = await repo.findById(id);
    if (user == null) throw StateError('User $id not found');
    return user.name;
  }

  Future<User> createUser(int id, String name) async {
    final existing = await repo.findById(id);
    if (existing != null) throw StateError('User $id already exists');
    final user = User(id, name);
    await repo.save(user);
    return user;
  }
}

// --- Mock ---
class MockUserRepository implements UserRepository {
  final Map<int, User> _store;
  final List<User> savedUsers = [];

  MockUserRepository([Map<int, User>? initialData])
      : _store = Map.of(initialData ?? {});

  @override
  Future<User?> findById(int id) async => _store[id];

  @override
  Future<void> save(User user) async {
    _store[user.id] = user;
    savedUsers.add(user);
  }
}

// --- Tests ---
void main() {
  late MockUserRepository repo;
  late UserService service;

  setUp(() {
    repo = MockUserRepository({1: User(1, 'Alice')});
    service = UserService(repo);
  });

  group('getUserName', () {
    test('returns name of existing user', () async {
      expect(await service.getUserName(1), equals('Alice'));
    });

    test('throws StateError for missing user', () async {
      await expectLater(
        () => service.getUserName(99),
        throwsA(isA<StateError>().having((e) => e.message, 'message', contains('99'))),
      );
    });
  });

  group('createUser', () {
    test('creates and saves new user', () async {
      final user = await service.createUser(2, 'Bob');
      expect(user.name, 'Bob');
      expect(repo.savedUsers, hasLength(1));
      expect(repo.savedUsers.first.id, 2);
    });

    test('throws StateError if user already exists', () async {
      await expectLater(
        () => service.createUser(1, 'Duplicate'),
        throwsA(isA<StateError>()),
      );
      expect(repo.savedUsers, isEmpty); // save не был вызван
    });
  });
}
```

Ручной mock (`implements` + в памяти хранилище) прост и не требует зависимостей. `having()` позволяет проверять не только тип исключения, но и конкретные свойства. Проверяй побочные эффекты (сохранения) через публичные поля или методы mock-объекта — это называется «state verification».
