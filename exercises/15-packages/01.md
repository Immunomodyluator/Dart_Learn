**pubspec.yaml:**

```yaml
name: my_utils
description: >-
  A collection of utility functions for strings and collections.
  Provides capitalize, truncate, chunked and other helpers.
version: 0.1.0
homepage: https://github.com/user/my_utils

environment:
  sdk: ">=3.0.0 <4.0.0"

dev_dependencies:
  test: ^1.0.0
```

**lib/my_utils.dart:**

```dart
export 'src/string_utils.dart';
export 'src/list_utils.dart';
```

**lib/src/string_utils.dart:**

````dart
/// Утилиты для работы со строками.
///
/// Пример использования:
/// ```dart
/// print(capitalize('hello')); // Hello
/// ```
@Category('String utilities')
library;

/// Делает первую букву заглавной.
///
/// Возвращает пустую строку если [s] пустая.
String capitalize(String s) {
  if (s.isEmpty) return '';
  return s[0].toUpperCase() + s.substring(1);
}
````

Исходный код пакета должен лежать в `lib/` (публичная часть) и `lib/src/` (внутренняя реализация). Только через `export` в barrel-файле можно сделать классы из `src/` частью публичного API. `CHANGELOG.md` обязателен для pub.dev — туда пишутся изменения по версиям в формате Keep a Changelog.
