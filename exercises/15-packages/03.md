**lib/src/truncate.dart:**
```dart
/// Усекает строку [s] до [maxLength] символов.
///
/// Если длина [s] не превышает [maxLength], возвращает [s] без изменений.
/// Иначе обрезает строку и добавляет [ellipsis] (по умолчанию `'...'`).
///
/// Пример:
/// ```dart
/// truncate('Hello, world!', 5);          // 'He...'
/// truncate('Hello, world!', 5, ellipsis: '…'); // 'Hell…'
/// truncate('Hi', 10);                    // 'Hi'
/// ```
///
/// Бросает [ArgumentError] если [maxLength] <= 0 или [maxLength] меньше длины [ellipsis].
String truncate(String s, int maxLength, {String ellipsis = '...'}) {
  if (maxLength <= 0) throw ArgumentError.value(maxLength, 'maxLength', 'must be > 0');
  if (maxLength < ellipsis.length) {
    throw ArgumentError('maxLength ($maxLength) must be >= ellipsis.length (${ellipsis.length})');
  }
  if (s.length <= maxLength) return s;
  return s.substring(0, maxLength - ellipsis.length) + ellipsis;
}
```

**lib/src/email.dart:**
```dart
/// Проверяет базовый формат email.
///
/// Возвращает `true` если строка содержит символы до `@`,
/// один символ `@`, домен с точкой и хотя бы двумя символами TLD.
///
/// **Примечание:** это базовая проверка форматa, не RFC 5321 валидация.
///
/// Пример:
/// ```dart
/// isEmail('user@example.com');  // true
/// isEmail('invalid-email');     // false
/// isEmail('a@b.c');             // true
/// ```
bool isEmail(String s) {
  final emailRegex = RegExp(r'^[^@\s]+@[^@\s]+\.[^@\s]{2,}$');
  return emailRegex.hasMatch(s);
}
```
