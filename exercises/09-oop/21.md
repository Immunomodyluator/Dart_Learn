Перепиши код с использованием каскадного оператора `..` и null-aware каскада `?..`.

```dart
// До — без каскадов
void withoutCascade() {
  final sb = StringBuffer();
  sb.write('Hello');
  sb.write(', ');
  sb.write('World');
  sb.writeln('!');
  print(sb.toString());

  List<int>? nums;
  if (nums != null) {
    nums.add(1);
    nums.add(2);
    nums.sort();
  }
}

// После — с каскадами (напиши здесь)
void withCascade() {
  // ...
}

void main() => withCascade();
```

<details>
<summary>Решение</summary>

```dart
void withCascade() {
  // Каскад на StringBuffer
  final sb = StringBuffer()
    ..write('Hello')
    ..write(', ')
    ..write('World')
    ..writeln('!');
  print(sb.toString()); // Hello, World!\n

  // Null-aware каскад
  List<int>? nums;
  nums
    ?..add(1)
    ?..add(2)
    ?..sort();
  // nums == null — ничего не произошло, нет NPE

  nums = [3, 1, 2];
  nums
    ..add(0)
    ..sort();
  print(nums); // [0, 1, 2, 3]
}

void main() => withCascade();
```

`..` — каскадный оператор: применяет метод/установку свойства к объекту и **возвращает тот же объект**, позволяя цепочку. `?..` — null-aware версия: пропускает остаток цепочки если объект `null`. Каскады особенно удобны для builder-паттерна и инициализации объектов.
