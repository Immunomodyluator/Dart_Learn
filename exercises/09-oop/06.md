Код не компилируется. Найди и исправь все ошибки.

```dart
abstract class Logger {
  void log(String message);  // абстрактный метод

  void logError(String msg) => log('[ERROR] $msg'); // конкретный
}

class ConsoleLogger extends Logger {
  // log не переопределён — ошибка компиляции
}

class FileLogger extends Logger {
  final String path;
  FileLogger(this.path);

  @override
  void log(String message) => print('$path: $message');
}

void main() {
  final Logger l = Logger();  // нельзя создать abstract class
  l.log('test');
}
```

<summary>Решение</summary>

```dart
abstract class Logger {
  void log(String message);
  void logError(String msg) => log('[ERROR] $msg');
}

class ConsoleLogger extends Logger {
  @override
  void log(String message) => print(message); // реализовали абстрактный метод
}

class FileLogger extends Logger {
  final String path;
  FileLogger(this.path);

  @override
  void log(String message) => print('$path: $message');
}

void main() {
  // Logger() — нельзя; используем конкретные реализации
  final Logger l = ConsoleLogger();
  l.log('test');              // test
  l.logError('oops');        // [ERROR] oops

  final fl = FileLogger('/var/log/app.log');
  fl.logError('disk full');  // /var/log/app.log: [ERROR] disk full
}
```

**Два бага:** 1) `ConsoleLogger` не реализовал абстрактный метод `log` — обязательно для не-абстрактного подкласса. 2) `Logger()` — попытка создать экземпляр абстрактного класса, что запрещено; нужна конкретная реализация.
