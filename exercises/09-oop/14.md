# Задание 14 (9.7): Operators — 🐛 Найти баг

**Тема:** 09. ООП | **Подтема:** 9.7 Перегрузка операторов

---

Класс `Money` ведёт себя неправильно при сравнении и хранении в `Set`. Найди и исправь все ошибки.

```dart
class Money {
  final int cents;
  final String currency;

  Money(this.cents, this.currency);

  Money operator +(Money other) {
    if (currency != other.currency) throw StateError('Currency mismatch');
    return Money(cents + other.cents, currency);
  }

  // Переопределён == но забыт hashCode
  @override
  bool operator ==(Object other) =>
      other is Money && cents == other.cents && currency == other.currency;

  @override
  String toString() => '${(cents / 100).toStringAsFixed(2)} $currency';
}

void main() {
  final a = Money(100, 'USD');
  final b = Money(100, 'USD');

  print(a == b);                   // true  — ok
  final s = {a, b};
  print(s.length);                 // ожидается 1, получается 2
}
```

<summary>Решение</summary>

```dart
class Money {
  final int cents;
  final String currency;

  const Money(this.cents, this.currency);

  Money operator +(Money other) {
    if (currency != other.currency) throw StateError('Currency mismatch');
    return Money(cents + other.cents, currency);
  }

  @override
  bool operator ==(Object other) =>
      other is Money && cents == other.cents && currency == other.currency;

  @override
  int get hashCode => Object.hash(cents, currency); // добавлен hashCode

  @override
  String toString() => '${(cents / 100).toStringAsFixed(2)} $currency';
}

void main() {
  final a = Money(100, 'USD');
  final b = Money(100, 'USD');

  print(a == b);       // true
  final s = {a, b};
  print(s.length);     // 1
}
```

**Баг:** `operator ==` переопределён, но `hashCode` оставлен по умолчанию (`identityHashCode`). Контракт Dart требует: если `a == b`, то `a.hashCode == b.hashCode`. `Set` и `Map` используют `hashCode` для определения бакета — без совпадения хешей объекты попадают в разные бакеты и считаются разными.
