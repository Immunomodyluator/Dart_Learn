# Задание 11 (9.6): Extension methods — 🖊 Написать код

**Тема:** 09. ООП | **Подтема:** 9.6 Extension methods

---

Напиши extension-методы для стандартных типов:

1. `extension StringExtensions on String`:
   - `bool get isBlank` — пустая или содержит только пробелы.
   - `String capitalize()` — первая буква заглавная, остальные как есть.

2. `extension ListExtensions<T> on List<T>`:
   - `List<List<T>> chunked(int size)` — разбивает список на части по `size` элементов.

<details>
<summary>Решение</summary>

```dart
extension StringExtensions on String {
  bool get isBlank => trim().isEmpty;

  String capitalize() {
    if (isEmpty) return this;
    return this[0].toUpperCase() + substring(1);
  }
}

extension ListExtensions<T> on List<T> {
  List<List<T>> chunked(int size) {
    assert(size > 0, 'size must be positive');
    final result = <List<T>>[];
    for (var i = 0; i < length; i += size) {
      result.add(sublist(i, (i + size).clamp(0, length)));
    }
    return result;
  }
}

void main() {
  print(''.isBlank);          // true
  print('  '.isBlank);        // true
  print('hello'.isBlank);     // false
  print('dart'.capitalize()); // Dart

  print([1, 2, 3, 4, 5].chunked(2)); // [[1, 2], [3, 4], [5]]
  print([1, 2, 3].chunked(5));        // [[1, 2, 3]]
}
```

Extension-методы добавляют методы к существующим типам без изменения их исходного кода и без наследования. Они разрешаются статически — нет полиморфизма. В generics-extension `<T>` повторяет параметр типа класса. `(i + size).clamp(0, length)` безопасно ограничивает верхнюю границу `sublist`.
