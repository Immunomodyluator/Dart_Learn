```dart
class Point {
  final double x;
  final double y;

  Point(double x, double y) {  // поля не инициализируются
    x = this.x;   // перепутаны стороны присваивания
    y = this.y;
  }

  Point.origin()  // именованный конструктор без тела/инициализатора

  double distanceTo(Point other) {
    final dx = x - other.x;
    final dy = y - other.y;
    return (dx * dx + dy * dy).sqrt();  // нет такого метода
  }
}

void main() {
  final p = Point(3, 4);
  final o = Point.origin();
  print(p.distanceTo(o)); // ожидается 5.0
}
```

<summary>Решение</summary>

```dart
import 'dart:math';

class Point {
  final double x;
  final double y;

  Point(this.x, this.y);      // инициализирующие параметры

  Point.origin() : x = 0, y = 0; // именованный конструктор с инициализатором

  double distanceTo(Point other) {
    final dx = x - other.x;
    final dy = y - other.y;
    return sqrt(dx * dx + dy * dy); // sqrt из dart:math
  }
}

void main() {
  final p = Point(3, 4);
  final o = Point.origin();
  print(p.distanceTo(o)); // 5.0
}
```

**Три бага:** 1) `x = this.x` — присваивание идёт в обратную сторону; правильно `this.x = x`, но проще `Point(this.x, this.y)`. 2) `Point.origin()` без тела/инициализатора — синтаксическая ошибка; добавлен инициализатор `: x = 0, y = 0`. 3) `.sqrt()` — нет такого метода на `double`; используй `sqrt(...)` из `dart:math`.
