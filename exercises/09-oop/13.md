# Задание 13 (9.7): Operators — 🖊 Написать код

**Тема:** 09. ООП | **Подтема:** 9.7 Перегрузка операторов

---

Реализуй класс `Vector2D` с перегруженными операторами:

- Поля `double x`, `double y`.
- `operator +` — сложение векторов.
- `operator *` (скаляр) — умножение на число.
- `operator ==` и `hashCode` — сравнение по значению.
- `@override toString()` → `'Vector2D($x, $y)'`.

<details>
<summary>Решение</summary>

```dart
class Vector2D {
  final double x;
  final double y;

  const Vector2D(this.x, this.y);

  Vector2D operator +(Vector2D other) => Vector2D(x + other.x, y + other.y);
  Vector2D operator *(double scalar)  => Vector2D(x * scalar,  y * scalar);
  Vector2D operator -()               => Vector2D(-x, -y);   // унарный минус

  @override
  bool operator ==(Object other) =>
      other is Vector2D && x == other.x && y == other.y;

  @override
  int get hashCode => Object.hash(x, y);

  @override
  String toString() => 'Vector2D($x, $y)';
}

void main() {
  final v1 = Vector2D(1, 2);
  final v2 = Vector2D(3, 4);

  print(v1 + v2);    // Vector2D(4.0, 6.0)
  print(v1 * 3);     // Vector2D(3.0, 6.0)
  print(-v2);        // Vector2D(-3.0, -4.0)
  print(v1 == Vector2D(1, 2)); // true
  print(v1 == v2);             // false
}
```

`operator +` — бинарный оператор с параметром. Унарный минус объявляется как `operator -()` без параметров. Переопределение `==` **всегда** требует переопределения `hashCode` — иначе объекты могут быть логически равны, но `Set`/`Map` считает их разными.
