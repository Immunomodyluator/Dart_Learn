# Задание 05 (9.3): Абстрактные классы — 🖊 Написать код

**Тема:** 09. ООП | **Подтема:** 9.3 Абстрактные классы

---

Реализуй абстрактную иерархию геометрических фигур:

- `abstract class Shape` — абстрактные геттеры `double area` и `double perimeter`; конкретный метод `void describe()` → печатает `'Shape: area=$area, perimeter=$perimeter'`.
- `class Circle extends Shape` — поле `double radius`.
- `class Rectangle extends Shape` — поля `double width`, `double height`.

Используй `dart:math` для π.

<details>
<summary>Решение</summary>

```dart
import 'dart:math';

abstract class Shape {
  double get area;
  double get perimeter;

  void describe() {
    print('Shape: area=${area.toStringAsFixed(2)}, '
        'perimeter=${perimeter.toStringAsFixed(2)}');
  }
}

class Circle extends Shape {
  final double radius;
  Circle(this.radius);

  @override
  double get area => pi * radius * radius;

  @override
  double get perimeter => 2 * pi * radius;
}

class Rectangle extends Shape {
  final double width, height;
  Rectangle(this.width, this.height);

  @override
  double get area => width * height;

  @override
  double get perimeter => 2 * (width + height);
}

void main() {
  final shapes = <Shape>[Circle(5), Rectangle(4, 6)];
  for (final s in shapes) s.describe();
  // Shape: area=78.54, perimeter=31.42
  // Shape: area=24.00, perimeter=20.00
}
```

`abstract class` нельзя инстанциировать напрямую. Абстрактный getter без тела обязывает подклассы предоставить реализацию. Конкретный метод `describe` в базовом классе использует `area` и `perimeter` полиморфно — подкласс определяет их значения.

</details>
