# Задание 09 (9.5): Mixin — 🖊 Написать код

**Тема:** 09. ООП | **Подтема:** 9.5 Mixin

---

Напиши миксины для добавления поведения к классам:

1. `mixin Flyable` — метод `void fly()` → `'$runtimeType is flying'`.
2. `mixin Swimmable` — метод `void swim()` → `'$runtimeType is swimming'`.
3. `class Bird` — использует `Flyable`.
4. `class Duck extends Bird with Swimmable` — использует оба.
5. `class Fish` — использует только `Swimmable`.

<details>
<summary>Решение</summary>

```dart
mixin Flyable {
  void fly() => print('$runtimeType is flying');
}

mixin Swimmable {
  void swim() => print('$runtimeType is swimming');
}

class Bird with Flyable {
  final String name;
  Bird(this.name);
}

class Duck extends Bird with Swimmable {
  Duck(super.name);
}

class Fish with Swimmable {
  final String species;
  Fish(this.species);
}

void main() {
  final bird = Bird('Eagle');
  bird.fly();           // Bird is flying

  final duck = Duck('Donald');
  duck.fly();           // Duck is flying
  duck.swim();          // Duck is swimming

  final fish = Fish('Salmon');
  fish.swim();          // Fish is swimming

  // Полиморфизм через mixin-тип
  final swimmers = <Swimmable>[duck, fish];
  for (final s in swimmers) s.swim();
}
```

`mixin` — способ добавить реализацию методов без наследования. `with Mixin1, Mixin2` применяет миксины слева направо (последний побеждает при конфликте имён). `mixin on BaseClass` ограничивает применение определённым базовым классом. Миксины не имеют конструкторов.

</details>
