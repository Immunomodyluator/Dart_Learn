# Задание 18 (9.9): Getters и setters — 🐛 Найти баг

**Тема:** 09. ООП | **Подтема:** 9.9 Getters и setters

---

Геттер и сеттер работают некорректно. Найди и исправь все ошибки.

```dart
class Circle {
  double radius;

  Circle(this.radius);

  // Геттер вызывает сам себя рекурсивно
  double get area => area;

  // Сеттер не изменяет backing field
  set radius(double r) {
    if (r < 0) throw ArgumentError('Radius cannot be negative');
    radius = r;   // рекурсивный вызов сеттера
  }

  // Геттер для diameter без backing field — ok?
  double get diameter => radius * 2;
  set diameter(double d) => radius = d;  // это нормально
}

void main() {
  final c = Circle(5);
  print(c.area);      // ожидается ~78.5
  c.radius = 10;
  print(c.diameter);  // ожидается 20
}
```

<details>
<summary>Решение</summary>

```dart
import 'dart:math';

class Circle {
  double _radius;  // приватный backing field

  Circle(double radius) : _radius = radius;

  double get radius => _radius;
  set radius(double r) {
    if (r < 0) throw ArgumentError('Radius cannot be negative');
    _radius = r;   // записываем в _radius, не в radius
  }

  double get area => pi * _radius * _radius; // вычисляется, не рекурсия

  double get diameter => _radius * 2;
  set diameter(double d) => radius = d / 2;  // исправлено: d/2, а не d
}

void main() {
  final c = Circle(5);
  print(c.area.toStringAsFixed(2)); // 78.54
  c.radius = 10;
  print(c.diameter);                // 20.0
  c.diameter = 14;
  print(c.radius);                  // 7.0
}
```

**Два бага:** 1) `get area => area` — геттер вызывает сам себя → бесконечная рекурсия (`StackOverflow`). Нужна формула `pi * _radius * _radius`. 2) `set radius(r) { radius = r; }` — `radius` в правой части снова обращается к сеттеру → бесконечная рекурсия. Нужно приватное поле `_radius` и запись `_radius = r`.
