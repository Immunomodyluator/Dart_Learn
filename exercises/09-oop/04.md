Код ведёт себя неожиданно: метод базового класса не вызывается, а супер-конструктор вызывается неверно. Найди и исправь все ошибки.

```dart
class Vehicle {
  final String brand;
  int speed = 0;

  Vehicle(this.brand);

  void accelerate(int delta) {
    speed += delta;
    print('$brand accelerates to $speed km/h');
  }
}

class Car extends Vehicle {
  final int doors;

  Car(String brand, this.doors) : super();  // super() без аргумента

  @override
  void accelerate(int delta) {
    speed += delta * 2;                      // удвоение без вызова super
    // хотим также вывести родительское сообщение
  }
}

void main() {
  final c = Car('Toyota', 4);
  c.accelerate(30); // ожидается сообщение + скорость 60
  print(c.speed);   // 60
}
```

<summary>Решение</summary>

```dart
class Vehicle {
  final String brand;
  int speed = 0;

  Vehicle(this.brand);

  void accelerate(int delta) {
    speed += delta;
    print('$brand accelerates to $speed km/h');
  }
}

class Car extends Vehicle {
  final int doors;

  Car(String brand, this.doors) : super(brand); // передаём brand в super

  @override
  void accelerate(int delta) {
    super.accelerate(delta * 2); // делегируем в базовый (обновит speed и напечатает)
  }
}

void main() {
  final c = Car('Toyota', 4);
  c.accelerate(30); // Toyota accelerates to 60 km/h
  print(c.speed);   // 60
}
```

**Два бага:** 1) `super()` вызывает конструктор `Vehicle` без аргументов, но `Vehicle(this.brand)` требует `brand`; компилятор выдаёт ошибку — нужно `super(brand)`. 2) В переопределении `accelerate` напрямую `speed += delta * 2` дублирует логику и не вызывает вывод из родителя; правильно делегировать через `super.accelerate(delta * 2)`.
