# Задание 17 (9.9): Getters и setters — 🖊 Написать код

**Тема:** 09. ООП | **Подтема:** 9.9 Getters и setters

---

Реализуй класс `Temperature` с умными геттерами и сеттерами:

- Хранит температуру в **Кельвинах** (`double _kelvin`).
- Getter/setter `double get celsius` / `set celsius(double)` — конвертация ↔ Кельвины.
- Getter/setter `double get fahrenheit` / `set fahrenheit(double)`.
- Setter должен бросать `ArgumentError` при установке ниже абсолютного нуля (0 K).

<details>
<summary>Решение</summary>

```dart
class Temperature {
  double _kelvin;

  Temperature({double kelvin = 273.15}) : _kelvin = kelvin {
    if (kelvin < 0) throw ArgumentError('Temperature below absolute zero');
  }

  double get kelvin => _kelvin;
  set kelvin(double k) {
    if (k < 0) throw ArgumentError('Temperature below absolute zero: $k K');
    _kelvin = k;
  }

  double get celsius => _kelvin - 273.15;
  set celsius(double c) => kelvin = c + 273.15;

  double get fahrenheit => celsius * 9 / 5 + 32;
  set fahrenheit(double f) => celsius = (f - 32) * 5 / 9;

  @override
  String toString() =>
      '${celsius.toStringAsFixed(2)}°C / ${fahrenheit.toStringAsFixed(2)}°F / ${_kelvin.toStringAsFixed(2)} K';
}

void main() {
  final t = Temperature(kelvin: 373.15);
  print(t); // 100.00°C / 212.00°F / 373.15 K

  t.celsius = 0;
  print(t); // 0.00°C / 32.00°F / 273.15 K

  t.fahrenheit = 98.6;
  print(t.celsius.toStringAsFixed(1)); // 37.0

  try {
    t.kelvin = -1;
  } on ArgumentError catch (e) {
    print(e.message); // Temperature below absolute zero: -1.0 K
  }
}
```

Геттер и сеттер позволяют контролировать доступ к данным, конвертируя или валидируя значения прозрачно для вызывающего кода. Хранение в одной единице (кельвины) и конвертация на лету — классический пример computed property.
