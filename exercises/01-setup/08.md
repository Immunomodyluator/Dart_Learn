# Задание 08 (1.4): Форматирование, анализ и линтеры — 🖊 Написать код

**Тема:** 01. Установка и инструменты | **Подтема:** 1.4 Форматирование, анализ и линтеры

---

Напиши команду `dart format`, которая:

1. Проверяет форматирование **без изменения** файлов
2. Завершается с ненулевым кодом, если код не отформатирован
3. Применяется ко всем файлам в текущей директории (рекурсивно)

Команда будет использоваться в CI для проверки стиля.

<details>
<summary>Решение</summary>

```bash
dart format --output=none --set-exit-if-changed .
```

- `--output=none` — не записывать изменения в файлы (только проверка).
- `--set-exit-if-changed` — вернуть exit code `1`, если хотя бы один файл нуждается в форматировании.
- `.` — текущая директория рекурсивно.

CI увидит ненулевой код выхода и отметит шаг как неуспешный.

</details>
