# Задание 07 (1.4): Форматирование, анализ и линтеры — 🔲 Дополнить код

---

Дополни файл `analysis_options.yaml`: подключи пакет `lints` (рекомендуемые правила) и включи правило `always_declare_return_types`.

```yaml
include: ___

linter:
  rules:
    ___: ___
```

<details>
<summary>Решение</summary>

```yaml
include: package:lints/recommended.yaml

linter:
  rules:
    always_declare_return_types: true
```

`include: package:lints/recommended.yaml` подключает набор правил от команды Dart. Дополнительные правила включаются в `linter.rules`. Не забудь добавить `lints` в `dev_dependencies` в `pubspec.yaml`.

</details>
