**Тема:** 01. Установка и инструменты | **Подтема:** 1.1 Установка SDK и редакторов

---

Разработчик хочет проверить версию установленного Dart SDK, но команда не работает. Найди и исправь ошибку.

```bash
dart -version
```

Ожидаемый вывод:
```
Dart SDK version: 3.x.x (stable)
```

<details>
<summary>Решение</summary>

```bash
dart --version
```

В Dart CLI флаги пишутся с двойным дефисом: `--version`, а не `-version`. Флаг с одним дефисом — POSIX-стиль для однобуквенных флагов (`-v`), Dart SDK его здесь не поддерживает.
</details>
