Разработчик пытается скомпилировать программу в нативный исполняемый файл, но получает ошибку. Найди все проблемы в команде.

```bash
dart compile bin/server.dart --output build/server
```

<details>
<summary>Решение</summary>

```bash
dart compile exe bin/server.dart -o build/server
```

Два исправления:
1. Нужно указать формат компиляции — `exe` (нативный исполняемый файл); без него команда не знает, что нужно сгенерировать.
2. `--output` — неверное имя флага; правильно: `-o` (или `--output` существует в некоторых подкомандах, но для `compile exe` нужен `-o`).
</details>
