# Задание 03 (1.2): Dart CLI и dartdev инструменты — 🖊 Написать код

**Тема:** 01. Установка и инструменты | **Подтема:** 1.2 Dart CLI и dartdev инструменты

---

Напиши файл `bin/hello.dart` с программой, которая выводит `Hello, Dart!`, и укажи правильную команду для её **запуска без предварительной компиляции** через Dart CLI.

<details>
<summary>Решение</summary>

```dart
// bin/hello.dart
void main() {
  print('Hello, Dart!');
}
```

```bash
dart run bin/hello.dart
```

`dart run` запускает файл через JIT-компилятор напрямую — компиляция не нужна. Для скриптов из `bin/` с прописанным `executables` в `pubspec.yaml` работает сокращённая форма `dart run`.
</details>
