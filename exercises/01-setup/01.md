# Задание 01 (1.1): Установка SDK и редакторов — 🔲 Дополнить код

**Тема:** 01. Установка и инструменты | **Подтема:** 1.1 Установка SDK и редакторов

---

Дополни файл `pubspec.yaml` для нового Dart-проекта. Заполни все пропуски (`___`), чтобы файл был корректным и прошёл `dart pub get`.

```yaml
___ : my_first_app

___ : 1.0.0

description: My first Dart application.

environment:
  sdk: '___'

dependencies:
  ___:
```

<details>
<summary>Решение</summary>

```yaml
name: my_first_app

version: 1.0.0

description: My first Dart application.

environment:
  sdk: '>=3.0.0 <4.0.0'

dependencies:
  # пусто — зависимостей нет
```

`name` и `version` — обязательные поля. `environment.sdk` задаёт ограничение на версию SDK. Пустой блок `dependencies` можно опустить, но для наглядности часто оставляют.
</details>
