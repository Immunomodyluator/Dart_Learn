# Задание 05 (22.3): Безопасность при взаимодействии с нативом/JS — 🖊 Написать код

**Тема:** 22. Безопасность и защита кода | **Подтема:** 22.3 Безопасность при взаимодействии с нативом/JS

---

Реализуй безопасные обёртки для работы с нативными функциями (FFI) и JS-интеропом.

**Требования:**

1. **FFI — безопасная обёртка над `system()`** — нативная функция `system(cmd)` выполняет shell-команды. Напиши обёртку `safeSystem(String command)`, которая:
   - Принимает только команду из белого списка (`['ls', 'pwd', 'whoami']`)
   - Запрещает аргументы с метасимволами shell (`;`, `|`, `&`, `$`, `` ` ``)
   - Освобождает память через `calloc.free` в `finally`
   - Возвращает `int` (exit code) или выбрасывает `SecurityException`

2. **JS interop — безопасный доступ к localStorage** — используй `dart:js_interop`, напиши `SecureStorage` с методами `getItem(key)` / `setItem(key, value)`:
   - Валидирует ключ: только `[a-zA-Z0-9_-]`, максимум 64 символа
   - Валидирует значение: не более 4096 символов, без нулевых байтов
   - `getItem` возвращает `String?` (null если не найдено), не выбрасывает

3. Для каждой функции напиши небольшой сценарий использования с примерами безопасного и небезопасного ввода.

```dart
// ── 1. FFI: безопасная обёртка над system() ───────────────────────

import 'dart:ffi';
import 'package:ffi/ffi.dart';

class SecurityException implements Exception {
  final String message;
  const SecurityException(this.message);
  @override
  String toString() => 'SecurityException: $message';
}

// Тип нативной функции system(const char*) → int
typedef SystemC = Int32 Function(Pointer<Utf8>);
typedef SystemDart = int Function(Pointer<Utf8>);

// Белый список разрешённых команд (без аргументов)
const _allowedCommands = {'ls', 'pwd', 'whoami', 'date', 'hostname'};

// Паттерн запрещённых shell-метасимволов
final _shellMetachars = RegExp(r'[;&|$`\\!<>(){}\[\]]');

/// Безопасная обёртка над нативной system()
int safeSystem(String command) {
  // 1. Разрешаем только команды из белого списка
  final parts = command.trim().split(RegExp(r'\s+'));
  final executable = parts.first;
  if (!_allowedCommands.contains(executable)) {
    throw SecurityException(
      'Command "$executable" is not in the allowlist: $_allowedCommands',
    );
  }

  // 2. Проверяем аргументы на метасимволы
  for (final part in parts.skip(1)) {
    if (_shellMetachars.hasMatch(part)) {
      throw SecurityException(
        'Argument "$part" contains forbidden shell metacharacters',
      );
    }
  }

  // 3. Ограничиваем длину команды
  if (command.length > 256) {
    throw SecurityException('Command too long (max 256 chars)');
  }

  // 4. Выполняем через FFI с обязательным освобождением памяти
  final lib = DynamicLibrary.process();
  final systemFunc = lib.lookupFunction<SystemC, SystemDart>('system');

  final cmdPtr = command.toNativeUtf8(allocator: calloc);
  try {
    return systemFunc(cmdPtr);
  } finally {
    calloc.free(cmdPtr);  // всегда освобождаем память
  }
}

// Демонстрация FFI
void ffiDemo() {
  // Безопасный вызов
  try {
    final code = safeSystem('ls');
    print('ls exit code: $code');
  } on SecurityException catch (e) {
    print('Blocked: $e');
  }

  // Попытка инъекции
  try {
    safeSystem('ls; rm -rf /');  // содержит ';'
  } on SecurityException catch (e) {
    print('Blocked injection: $e');
    // SecurityException: Argument ";" contains forbidden shell metacharacters
  }

  // Неразрешённая команда
  try {
    safeSystem('curl https://evil.com');
  } on SecurityException catch (e) {
    print('Blocked command: $e');
    // SecurityException: Command "curl" is not in the allowlist
  }
}
```

```dart
// ── 2. JS Interop: безопасный доступ к localStorage ───────────────

// @JS-декларации (web-only)
import 'dart:js_interop';

@JS('localStorage')
external JSObject get _localStorage;

extension type _Storage(JSObject _) implements JSObject {
  external String? getItem(String key);
  external void setItem(String key, String value);
  external void removeItem(String key);
}

_Storage get _storage => _Storage(_localStorage);

// Паттерн для безопасных ключей
final _keyPattern = RegExp(r'^[a-zA-Z0-9_\-]{1,64}$');

/// Безопасная обёртка над localStorage
class SecureStorage {
  /// Получает значение по ключу. Возвращает null если не найдено или ключ невалиден.
  static String? getItem(String key) {
    if (!_isValidKey(key)) {
      // Не выбрасываем — предсказуемое поведение при невалидном ключе
      return null;
    }
    try {
      return _storage.getItem(key);
    } catch (_) {
      // localStorage может быть недоступен (private mode, SecurityError)
      return null;
    }
  }

  /// Устанавливает значение. Выбрасывает SecurityException при нарушении ограничений.
  static void setItem(String key, String value) {
    if (!_isValidKey(key)) {
      throw SecurityException(
        'Invalid key "$key": only [a-zA-Z0-9_-] allowed, max 64 chars',
      );
    }
    if (!_isValidValue(value)) {
      throw SecurityException(
        'Invalid value: must be ≤ 4096 chars and not contain null bytes',
      );
    }
    try {
      _storage.setItem(key, value);
    } catch (e) {
      throw SecurityException('localStorage.setItem failed: $e');
    }
  }

  /// Удаляет ключ.
  static void removeItem(String key) {
    if (!_isValidKey(key)) return;  // молча игнорируем невалидные ключи
    try {
      _storage.removeItem(key);
    } catch (_) {}
  }

  static bool _isValidKey(String key) => _keyPattern.hasMatch(key);

  static bool _isValidValue(String value) =>
      value.length <= 4096 && !value.contains('\x00');
}

// Демонстрация JS interop
void jsDemo() {
  // Безопасное использование
  SecureStorage.setItem('user_theme', 'dark');
  final theme = SecureStorage.getItem('user_theme');
  print('Theme: $theme');  // Theme: dark

  // Попытка XSS через ключ
  try {
    SecureStorage.setItem('<script>alert(1)</script>', 'value');
  } on SecurityException catch (e) {
    print('Blocked: $e');
    // Blocked: SecurityException: Invalid key "<script>...": only [a-zA-Z0-9_-] allowed
  }

  // Попытка null byte injection
  try {
    SecureStorage.setItem('key', 'value\x00injection');
  } on SecurityException catch (e) {
    print('Blocked null byte: $e');
  }

  // Слишком большое значение
  try {
    SecureStorage.setItem('big_key', 'x' * 5000);
  } on SecurityException catch (e) {
    print('Blocked oversized value: $e');
  }
}
```

Белый список команд FFI — самая надёжная защита от command injection: даже если атакующий контролирует ввод, он может использовать только заранее одобренные безвредные команды. Проверка метасимволов — дополнительный уровень защиты. `SecureStorage` применяет принцип defense-in-depth: валидация ключа предотвращает prototype pollution и XSS через DOM Storage API, а ограничение значения — атаки на размер квоты localStorage.
