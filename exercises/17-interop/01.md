# Задание 01 (17.1): Dart FFI — 🖊 Написать код

**Тема:** 17. Interop: FFI и Web | **Подтема:** 17.1 Dart FFI

---

Реализуй вызов функций из стандартной C-библиотеки через `dart:ffi`.

**Задача:**

Напиши Dart-программу, которая через FFI вызывает функции из стандартной libc:

1. `int abs(int n)` — модуль числа
2. `double sqrt(double x)` — квадратный корень (из libm)
3. `int strlen(Pointer<Char> str)` — длина C-строки

Для каждой функции:

- Объяви `typedef` для нативного типа и Dart-типа
- Загрузи библиотеку через `DynamicLibrary`
- Создай Dart-обёртку с удобным API

```dart
import 'dart:ffi';
import 'dart:io';
import 'package:ffi/ffi.dart'; // для Utf8/Char утилит

// TODO: реализуй вызов abs, sqrt, strlen через FFI
```

<details>
<summary>Решение</summary>

```dart
import 'dart:ffi';
import 'dart:io';
import 'package:ffi/ffi.dart';

// ─── 1. Typedefs ───────────────────────────────────────
// Нативные типы функций (C)
typedef NativeAbs = Int32 Function(Int32);
typedef NativeSqrt = Double Function(Double);
typedef NativeStrlen = Size Function(Pointer<Char>);

// Dart-типы функций
typedef DartAbs = int Function(int);
typedef DartSqrt = double Function(double);
typedef DartStrlen = int Function(Pointer<Char>);

// ─── 2. Загрузка библиотек ─────────────────────────────
DynamicLibrary _loadLibc() {
  if (Platform.isLinux || Platform.isAndroid) {
    return DynamicLibrary.open('libc.so.6');
  } else if (Platform.isMacOS || Platform.isIOS) {
    return DynamicLibrary.process(); // на macOS libc встроена в process
  } else if (Platform.isWindows) {
    return DynamicLibrary.open('msvcrt.dll');
  }
  throw UnsupportedError('Unsupported platform');
}

// ─── 3. Связывание функций ─────────────────────────────
final _libc = _loadLibc();

final _abs = _libc.lookupFunction<NativeAbs, DartAbs>('abs');
final _sqrt = _libc.lookupFunction<NativeSqrt, DartSqrt>('sqrt');
final _strlen = _libc.lookupFunction<NativeStrlen, DartStrlen>('strlen');

// ─── 4. Удобные Dart-обёртки ───────────────────────────
int absNative(int n) => _abs(n);

double sqrtNative(double x) {
  if (x < 0) throw ArgumentError('Cannot take sqrt of negative number');
  return _sqrt(x);
}

int strlenNative(String s) {
  // Выделяем C-строку в native heap
  final cStr = s.toNativeUtf8().cast<Char>();
  try {
    return _strlen(cStr);
  } finally {
    calloc.free(cStr); // освобождаем память!
  }
}

// ─── 5. Пример использования ───────────────────────────
void main() {
  print(absNative(-42));          // 42
  print(sqrtNative(2.0));         // 1.4142135623730951
  print(strlenNative('Hello'));   // 5
  print(strlenNative(''));        // 0
}
```

При работе с FFI всегда освобождай нативную память (`calloc.free`) в `finally`-блоке — утечки памяти здесь не управляются сборщиком мусора Dart. `DynamicLibrary.process()` на macOS возвращает символы из уже загруженных библиотек, включая libc. На Linux нативные библиотеки нужно загружать явно по имени файла.
