Создай модели данных с помощью `freezed` и `json_serializable`.

**Задача:**

Реализуй модель `ApiResponse<T>` для типовых ответов REST API. Требования:
- Неизменяемый (immutable) data class
- JSON сериализация через `json_serializable`
- `copyWith` метод
- Паттерн-матчинг через sealed-подходF
- Два варианта: `ApiResponse.success(data, statusCode)` и `ApiResponse.error(message, statusCode)`

**1.** Напиши модель с `@freezed`:

```dart
import 'package:freezed_annotation/freezed_annotation.dart';

part 'api_response.freezed.dart';
part 'api_response.g.dart';

// TODO: @freezed class ApiResponse<T> with _$ApiResponse<T>
```

**2.** Напиши в `main` пример использования: создай `success`, `error`, обработай через `.when()`, скопируй через `copyWith`, сериализуй в JSON.

<details>
<summary>Решение</summary>

```dart
// lib/models/api_response.dart
import 'package:freezed_annotation/freezed_annotation.dart';

part 'api_response.freezed.dart';
part 'api_response.g.dart';

@freezed
class ApiResponse<T> with _$ApiResponse<T> {
  // Успешный ответ
  const factory ApiResponse.success({
    required T data,
    @Default(200) int statusCode,
  }) = ApiSuccess<T>;

  // Ответ с ошибкой
  const factory ApiResponse.error({
    required String message,
    @Default(500) int statusCode,
    String? errorCode,
  }) = ApiError<T>;

  // JSON сериализация требует фабрику fromJson
  factory ApiResponse.fromJson(
    Map<String, dynamic> json,
    T Function(Object? json) fromJsonT,
  ) => _$ApiResponseFromJson(json, fromJsonT);
}
```

```dart
// Пример использования (без реального запуска генерации)
// main.dart
import 'dart:convert';

void main() {
  // 1. Создание
  const response = ApiResponse<String>.success(data: 'Hello', statusCode: 200);
  const errResponse = ApiResponse<String>.error(
    message: 'Not Found',
    statusCode: 404,
    errorCode: 'ERR_404',
  );

  // 2. Pattern matching через .when()
  final msg = response.when(
    success: (data, statusCode) => 'OK $statusCode: $data',
    error: (msg, statusCode, code) => 'Error $statusCode: $msg',
  );
  print(msg); // OK 200: Hello

  // 3. .maybeWhen() — только один случай
  errResponse.maybeWhen(
    error: (msg, code, errCode) => print('Failed: $msg'),
    orElse: () => print('Success'),
  );

  // 4. copyWith
  final updated = response.copyWith(statusCode: 201);
  print(updated); // ApiSuccess(data: Hello, statusCode: 201)

  // 5. Сравнение по значению (==  реализован freezed)
  const same = ApiResponse<String>.success(data: 'Hello', statusCode: 200);
  print(response == same); // true

  // 6. Тип-проверка (is)
  if (errResponse is ApiError) {
    print('Error code: ${errResponse.errorCode}');
  }
}
```

**pubspec.yaml:**
```yaml
dependencies:
  freezed_annotation: ^2.4.0
  json_annotation: ^4.8.0

dev_dependencies:
  build_runner: ^2.4.0
  freezed: ^2.5.0
  json_serializable: ^6.7.0
```

`freezed` генерирует: `copyWith`, `==`, `hashCode`, `toString`, `map`/`when`/`maybeWhen` для union-типов. Для JSON сериализации `freezed`-класс комбинируется с `json_serializable` через два `part` файла. `@Default(value)` устанавливает дефолтное значение при десериализации.
