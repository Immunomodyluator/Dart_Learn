# Задание 06 (7.3): Стрелочные функции и анонимные функции — 🐛 Найти баг

**Тема:** 07. Функции | **Подтема:** 7.3 Стрелочные функции и анонимные функции

---

Код не компилируется или ведёт себя неожиданно. Найди и исправь все ошибки.

```dart
void main() {
  // 1. Стрелочная функция с блоком
  final double area = (double r) => {
    return 3.14 * r * r;
  };

  // 2. Тип не совпадает
  int Function(int) double = (x) => x * 2;

  // 3. Пустая стрелочная функция
  final noop = () => {};

  print(area(5));
  print(double(4));
  noop();
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  // 1. => и блок несовместимы — убрать фигурные скобки и return
  final area = (double r) => 3.14 * r * r;

  // 2. double — зарезервированный тип в Dart, нельзя использовать как имя
  int Function(int) twice = (x) => x * 2;

  // 3. () => {} возвращает Map<Never, Never>, а не void
  final noop = () {};   // блочная запись без return — правильно

  print(area(5));   // 78.5
  print(twice(4));  // 8
  noop();
}
```

**Три бага:** 1) `=> { return ...; }` — стрелочная функция не принимает блок; либо `=> expr`, либо `{ return expr; }`. 2) `double` — встроенный тип, нельзя использовать как имя переменной. 3) `() => {}` создаёт функцию, возвращающую пустой `Map`, а не `void`. Для void-замыкания используй `() {}` (блочную форму).
