# Задание 05 (7.3): Стрелочные функции и анонимные функции — 🔲 Дополнить код

**Тема:** 07. Функции | **Подтема:** 7.3 Стрелочные функции и анонимные функции

---

Перепиши каждую `function`-функцию в виде стрелочной (`=>`), сохраняя поведение.

```dart
// 1. Обычная функция → стрелочная
bool isPositive(int n) {
  return n > 0;
}
// bool isPositive(int n) ___ n > 0;

// 2. Анонимная в переменной → стрелочная
final double Function(double) square = (x) {
  return x * x;
};
// final square = (double x) ___ x * x;

// 3. Передача в sort → стрелочная лямбда
final nums = [3, 1, 4, 1, 5];
nums.sort((a, b) {
  return b.compareTo(a);  // убывание
});
// nums.sort((a, b) ___ b.compareTo(a));

void main() {
  print(isPositive(5));   // true
  print(square(4));       // 16.0
  print(nums);            // [5, 4, 3, 1, 1]
}
```

<details>
<summary>Решение</summary>

```dart
bool isPositive(int n) => n > 0;

final square = (double x) => x * x;

void main() {
  final nums = [3, 1, 4, 1, 5];
  nums.sort((a, b) => b.compareTo(a));

  print(isPositive(5)); // true
  print(square(4));     // 16.0
  print(nums);          // [5, 4, 3, 1, 1]
}
```

`=> expr` — синтаксический сахар для `{ return expr; }`. Работает только для **одного** выражения. Не допускает `if-else` в качестве оператора (только условный оператор `? :`). Стрелочные функции делают код лаконичнее при передаче в `map`, `where`, `sort`, `forEach`.

</details>
