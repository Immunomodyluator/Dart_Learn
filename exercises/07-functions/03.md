# Задание 03 (7.2): Функции первого класса и замыкания — 🖊 Написать код

**Тема:** 07. Функции | **Подтема:** 7.2 Функции первого класса и замыкания

---

Напиши функцию `makeCounter({int start = 0, int step = 1})`, которая возвращает замыкание-счётчик. Каждый вызов возвращаемой функции увеличивает счётчик на `step` и возвращает новое значение.

Пример:

```
final counter = makeCounter(start: 10, step: 5);
print(counter()); // 15
print(counter()); // 20
print(counter()); // 25

final c2 = makeCounter();
print(c2()); // 1
print(c2()); // 2
```

<details>
<summary>Решение</summary>

```dart
int Function() makeCounter({int start = 0, int step = 1}) {
  var current = start;
  return () {
    current += step;
    return current;
  };
}

void main() {
  final counter = makeCounter(start: 10, step: 5);
  print(counter()); // 15
  print(counter()); // 20
  print(counter()); // 25

  final c2 = makeCounter();
  print(c2()); // 1
  print(c2()); // 2
}
```

Замыкание захватывает переменную `current` из окружающей области видимости. Каждый вызов `makeCounter` создаёт **отдельную** переменную `current` — поэтому `counter` и `c2` независимы. Тип возврата `int Function()` — функция без аргументов, возвращающая `int`.

</details>
