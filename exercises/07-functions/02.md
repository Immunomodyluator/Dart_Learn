# Задание 02 (7.1): Именованные и позиционные параметры — 🐛 Найти баг

**Тема:** 07. Функции | **Подтема:** 7.1 Именованные и позиционные параметры

---

Код не компилируется. Найди и исправь все ошибки, связанные с объявлением параметров.

```dart
// Нельзя смешивать необязательные позиционные и именованные
String describe(String name, [int? age], {String country = 'RU'}) {
  return '$name, $age, $country';
}

// Именованный без required и без значения по умолчанию
void connect({String host, int port = 5432}) {
  print('$host:$port');
}

void main() {
  print(describe('Alice', 25, country: 'US'));
  connect(host: 'localhost');
}
```

<details>
<summary>Решение</summary>

```dart
// Нельзя иметь одновременно необязательный позиционный [age] и именованный {country}
// Вариант 1: все именованные
String describe({required String name, int? age, String country = 'RU'}) {
  return '$name, $age, $country';
}

// Вариант 2: host помечается required
void connect({required String host, int port = 5432}) {
  print('$host:$port');
}

void main() {
  print(describe(name: 'Alice', age: 25, country: 'US'));
  connect(host: 'localhost');
}
```

**Два бага:** 1) Dart запрещает одновременно необязательные позиционные `[...]` и именованные `{...}` параметры — выбери один стиль. 2) Именованный параметр `{String host}` без `required` и без значения по умолчанию — это ошибка компиляции: нужно добавить `required` или `= 'localhost'`.

</details>
