# Задание 09 (7.5): typedef и функциональные типы — 🖊 Написать код

**Тема:** 07. Функции | **Подтема:** 7.5 typedef и функциональные типы

---

Напиши функциональный конвейер (pipeline) с помощью `typedef`:

1. Объяви `typedef Transformer<T> = T Function(T value)`.
2. Напиши функцию `T pipe<T>(T initial, List<Transformer<T>> transforms)`, которая последовательно применяет все трансформации.

Пример использования:
```
final result = pipe<String>(
  'hello world',
  [
    (s) => s.trim(),
    (s) => s.toUpperCase(),
    (s) => s.replaceAll(' ', '_'),
  ],
);
print(result); // HELLO_WORLD
```

<details>
<summary>Решение</summary>

```dart
typedef Transformer<T> = T Function(T value);

T pipe<T>(T initial, List<Transformer<T>> transforms) {
  return transforms.fold(initial, (value, fn) => fn(value));
}

void main() {
  final result = pipe<String>(
    '  hello world  ',
    [
      (s) => s.trim(),
      (s) => s.toUpperCase(),
      (s) => s.replaceAll(' ', '_'),
    ],
  );
  print(result); // HELLO_WORLD

  final num = pipe<int>(2, [(n) => n * 3, (n) => n + 1, (n) => n * n]);
  print(num);    // 49  (2*3=6, 6+1=7, 7*7=49)
}
```

`typedef` именует функциональный тип, делая объявления читаемыми. `fold` — свёртка: `initial` — начальный аккумулятор, лямбда применяет каждую трансформацию к текущему значению. Generic `<T>` делает `pipe` пригодным для любых типов.
</details>
