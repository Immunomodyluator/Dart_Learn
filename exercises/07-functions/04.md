# Задание 04 (7.2): Функции первого класса и замыкания — 🐛 Найти баг

**Тема:** 07. Функции | **Подтема:** 7.2 Функции первого класса и замыкания

---

Все элементы списка `actions` при вызове печатают одно и то же число. Объясни почему и исправь.

```dart
void main() {
  final actions = <void Function()>[];

  for (var i = 0; i < 3; i++) {
    actions.add(() => print(i));  // захват переменной i
  }

  for (final action in actions) {
    action(); // ожидается 0, 1, 2 — получается 3, 3, 3
  }
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  final actions = <void Function()>[];

  for (var i = 0; i < 3; i++) {
    final captured = i;           // создать новую переменную на каждой итерации
    actions.add(() => print(captured));
  }

  for (final action in actions) {
    action(); // 0  1  2
  }
}
```

**Баг закрытия:** все лямбды захватывают **одну и ту же** переменную `i`, а не её значение в момент создания. К тому моменту как лямбды выполняются, цикл уже завершился и `i == 3`. Решение: на каждой итерации создавать отдельную переменную `final captured = i` — замыкание тогда захватывает разные переменные с разными значениями.

</details>
