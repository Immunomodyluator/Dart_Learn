Дополни пропуски: объяви функцию с правильными видами параметров.

```dart
// 1. Все именованные, name обязателен, role опционален со значением по умолчанию 'user'
String makeProfile({___ String name, String ___ = 'user'}) {
  return '$name / $role';
}

// 2. Первый позиционный обязателен, второй позиционный опционален
String greet(String firstName, [String? lastName]) {
  return lastName != null ? '$firstName $lastName' : firstName;
}

// 3. Смешанный: позиционный + именованный обязательный
void log(String message, {___ String level}) {
  print('[$level] $message');
}

void main() {
  print(makeProfile(name: 'Alice'));             // Alice / user
  print(makeProfile(name: 'Bob', role: 'admin')); // Bob / admin
  print(greet('Ivan'));                          // Ivan
  print(greet('Ivan', 'Petrov'));                // Ivan Petrov
  log('start', level: 'INFO');                  // [INFO] start
}
```

<summary>Решение</summary>

```dart
String makeProfile({required String name, String role = 'user'}) {
  return '$name / $role';
}

String greet(String firstName, [String? lastName]) {
  return lastName != null ? '$firstName $lastName' : firstName;
}

void log(String message, {required String level}) {
  print('[$level] $message');
}

void main() {
  print(makeProfile(name: 'Alice'));              // Alice / user
  print(makeProfile(name: 'Bob', role: 'admin')); // Bob / admin
  print(greet('Ivan'));                           // Ivan
  print(greet('Ivan', 'Petrov'));                 // Ivan Petrov
  log('start', level: 'INFO');                   // [INFO] start
}
```

`required` — именованный параметр обязанный (без значения по умолчанию). `=` — значение по умолчанию `String role = 'user'`. `[T? param]` — необязательный позиционный в квадратных скобках. Смешивать именованные и необязательные позиционные нельзя.
