# Задание 08 (7.4): Рекурсия — 🐛 Найти баг

Функция должна суммировать все числа от 1 до n. Она падает с `Stack Overflow`. Найди и исправь все ошибки.

```dart
int sumTo(int n) {
  if (n = 0) return 0;    // базовый случай
  return n + sumTo(n);    // рекурсивный вызов
}

void main() {
  print(sumTo(100)); // ожидается 5050
  print(sumTo(0));   // ожидается 0
}
```

<details>
<summary>Решение</summary>

```dart
int sumTo(int n) {
  if (n == 0) return 0;       // == для сравнения, не =
  return n + sumTo(n - 1);    // n-1 двигает к базовому случаю
}

void main() {
  print(sumTo(100)); // 5050
  print(sumTo(0));   // 0
}
```

**Два бага:** 1) `if (n = 0)` — это **присваивание**, а не сравнение; `n` всегда становится 0, базовый случай не работает. Нужно `n == 0`. 2) `sumTo(n)` — вызов с тем же аргументом, бесконечная рекурсия; нужен `sumTo(n - 1)` для движения к базовому случаю.

</details>
