# Задание 07 (7.4): Рекурсия — 🖊 Написать код

**Тема:** 07. Функции | **Подтема:** 7.4 Рекурсия

---

Напиши две рекурсивные функции:

1. `int factorial(int n)` — факториал числа n (n! = n × (n-1) × ... × 1, 0! = 1). Бросай `ArgumentError` при `n < 0`.
2. `int fibonacci(int n)` — n-е число Фибоначчи (0-индексированное: fib(0)=0, fib(1)=1, fib(2)=1, fib(3)=2...). Бросай `ArgumentError` при `n < 0`.

Не используй итерации — только рекурсию.

<details>
<summary>Решение</summary>

```dart
int factorial(int n) {
  if (n < 0) throw ArgumentError('n must be >= 0, got $n');
  if (n == 0) return 1;
  return n * factorial(n - 1);
}

int fibonacci(int n) {
  if (n < 0) throw ArgumentError('n must be >= 0, got $n');
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

void main() {
  print(factorial(0)); // 1
  print(factorial(5)); // 120
  print(fibonacci(0)); // 0
  print(fibonacci(7)); // 13
}
```

Каждая рекурсивная функция имеет **базовый случай** (возврат без рекурсии) и **рекурсивный случай**. `factorial` — хвостовой рекурентный; `fibonacci` — дерево вызовов растёт экспоненциально (для больших `n` используй мемоизацию или итерацию). Dart не оптимизирует хвостовую рекурсию автоматически.

</details>
