# Задание 08 (2.4): const и final — 🐛 Найти баг

**Тема:** 02. Основы синтаксиса | **Подтема:** 2.4 const и final

---

Код не компилируется. Найди и исправь ошибку.

```dart
class Config {
  const Config(this.host, this.port);
  final String host;
  final int port;
}

void main() {
  const cfg = Config('localhost', DateTime.now().millisecondsSinceEpoch);
  print(cfg.host);
}
```

<details>
<summary>Решение</summary>

```dart
class Config {
  const Config(this.host, this.port);
  final String host;
  final int port;
}

void main() {
  // Вариант 1: final вместо const
  final cfg = Config('localhost', DateTime.now().millisecondsSinceEpoch);

  // Вариант 2: константные аргументы
  // const cfg = Config('localhost', 8080);

  print(cfg.host);
}
```

`const` требует, чтобы все аргументы конструктора были константами компиляции. `DateTime.now()` — рантаймовое значение, поэтому `const cfg = Config(...)` не компилируется. Решение: заменить `const` на `final`, или передать литерал вместо `DateTime.now()`.

</details>
