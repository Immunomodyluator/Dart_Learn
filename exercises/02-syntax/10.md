# Задание 10 (2.5): Null Safety — 🐛 Найти баг

**Тема:** 02. Основы синтаксиса | **Подтема:** 2.5 Null Safety

---

Код компилируется, но падает в рантайме при вызове с `null`. Найди небезопасное место и исправь его.

```dart
String getFirstWord(String? sentence) {
  return sentence!.split(' ').first;
}

void main() {
  print(getFirstWord('Hello World')); // Hello
  print(getFirstWord(null));          // ???
}
```

<details>
<summary>Решение</summary>

```dart
String? getFirstWord(String? sentence) {
  return sentence?.split(' ').first;
}

void main() {
  print(getFirstWord('Hello World')); // Hello
  print(getFirstWord(null));          // null
}
```

`sentence!` — force unwrap: бросает `Null check operator used on a null value` если `sentence == null`. Правильное решение: оператор `?.` (null-aware вызов) + тип возврата `String?`. Оператор `!` допустим только когда ты **гарантированно** знаешь, что значение не null.

</details>
