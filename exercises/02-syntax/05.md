# Задание 05 (2.3): Типы и вывод типов — 🖊 Написать код

**Тема:** 02. Основы синтаксиса | **Подтема:** 2.3 Типы и вывод типов

---

Напиши функцию `describe(Object? value)`, которая возвращает строку с описанием переданного значения:

- `"int: 42"` для целых чисел
- `"String: hello"` для строк
- `"bool: true"` для булевых
- `"other: double"` (с `runtimeType`) для остальных

Используй switch expression (Dart 3).

<details>
<summary>Решение</summary>

```dart
String describe(Object? value) {
  return switch (value) {
    int v    => 'int: $v',
    String v => 'String: $v',
    bool v   => 'bool: $v',
    _        => 'other: ${value.runtimeType}',
  };
}

void main() {
  print(describe(42));       // int: 42
  print(describe('hello'));  // String: hello
  print(describe(true));     // bool: true
  print(describe(3.14));     // other: double
}
```

Switch expression с pattern matching (Dart 3) — элегантный способ диспетчеризации по типу. Паттерн `int v` одновременно проверяет тип и связывает значение. `_` — wildcard для прочих случаев.

</details>
