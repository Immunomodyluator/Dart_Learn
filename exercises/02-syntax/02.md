Программа должна вывести три строки, но не компилируется. Найди и исправь все ошибки (их три).

```dart
void Main() {
  print('Line 1')
  print('Line 2');
  Print('Line 3');
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  print('Line 1');
  print('Line 2');
  print('Line 3');
}
```

Три ошибки:

1. `Main` → `main` — точка входа чувствительна к регистру.
2. Пропущена `;` после `print('Line 1')`.
3. `Print` → `print` — функции в Dart в `lowerCamelCase`.
</details>
