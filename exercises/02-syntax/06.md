Функция должна вернуть `true`, если сумма элементов списка больше 100, но содержит проблему с типами. Найди её и исправь.

```dart
bool isBigSum(List<int> numbers) {
  var sum = 0.0;
  for (var n in numbers) {
    sum += n;
  }
  return sum > 100;
}

void main() {
  print(isBigSum([40, 30, 40])); // ожидается: true
}
```

<details>
<summary>Решение</summary>

```dart
bool isBigSum(List<int> numbers) {
  var sum = 0; // было 0.0 — тип double, хотя всё — int
  for (var n in numbers) {
    sum += n;
  }
  return sum > 100;
}
```

Логика работает правильно, но тип `sum` — `double` из-за литерала `0.0`, хотя функция принимает `List<int>`. Это несогласованность типов, которую поймает линтер (`prefer_int_literals`). Правильно: `var sum = 0` → тип `int`.

</details>
