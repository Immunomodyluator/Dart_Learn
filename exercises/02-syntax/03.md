# Задание 03 (2.2): Объявление переменных — 🔲 Дополнить код

**Тема:** 02. Основы синтаксиса | **Подтема:** 2.2 Объявление переменных

---

Заполни пропуски (`___`): используй `var`, `final`, `dynamic` или явный тип согласно комментариям.

```dart
void main() {
  // Тип выводится, значение можно менять
  ___ counter = 0;
  counter = 10;

  // Значение устанавливается один раз в рантайме
  ___ greeting = 'Hello';

  // Может хранить любой тип
  ___ anything = 42;
  anything = 'now a string';

  // Явный тип — список строк, значение не меняется
  ___ List<String> names = ['Alice', 'Bob'];

  print('$counter $greeting $anything $names');
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  var counter = 0;
  counter = 10;

  final greeting = 'Hello';

  dynamic anything = 42;
  anything = 'now a string';

  final List<String> names = ['Alice', 'Bob'];

  print('$counter $greeting $anything $names');
}
```

`var` — тип выводится, значение можно менять. `final` — значение устанавливается один раз. `dynamic` — отключает статическую типизацию. Явный тип можно комбинировать с `final`.

</details>
