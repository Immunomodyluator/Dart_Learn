# Задание 07 (2.4): const и final — 🔲 Дополнить код

**Тема:** 02. Основы синтаксиса | **Подтема:** 2.4 const и final

---

Замени каждый `___` на `const` или `final` согласно комментариям.

```dart
void main() {
  // Значение известно на этапе компиляции
  ___ int maxRetries = 3;

  // Значение вычисляется в рантайме, не меняется
  ___ timestamp = DateTime.now();

  // Константный неизменяемый список
  ___ colors = ['red', 'green', 'blue'];

  print(maxRetries);
  print(timestamp);
  print(colors);
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  const int maxRetries = 3;

  final timestamp = DateTime.now();

  const colors = ['red', 'green', 'blue'];

  print(maxRetries);
  print(timestamp);
  print(colors);
}
```

`const` — значение должно быть известно на этапе компиляции (литерал или константное выражение). `DateTime.now()` вычисляется в рантайме → только `final`. `const` список автоматически неизменяемый: `colors.add(...)` бросит `UnsupportedError`.

</details>
