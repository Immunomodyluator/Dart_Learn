# Задание 01 (2.1): Hello World и структура программы — 🖊 Написать код

**Тема:** 02. Основы синтаксиса | **Подтема:** 2.1 Hello World и структура программы

---

Напиши программу, которая принимает имя через аргументы командной строки и выводит `Hello, <имя>!`. Если аргумент не передан — выводит `Hello, World!`.

<details>
<summary>Решение</summary>

```dart
void main(List<String> args) {
  final name = args.isNotEmpty ? args[0] : 'World';
  print('Hello, $name!');
}
```

`main` может принимать `List<String> args` — список аргументов командной строки. `args.isNotEmpty` проверяет, переданы ли аргументы. Запуск: `dart run bin/hello.dart Dart` выведет `Hello, Dart!`.

</details>
