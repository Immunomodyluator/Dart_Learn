# Задание 04 (2.2): Объявление переменных — 🐛 Найти баг

**Тема:** 02. Основы синтаксиса | **Подтема:** 2.2 Объявление переменных

---

Код не компилируется. Найди и объясни все ошибки типизации (их две).

```dart
void main() {
  var name = 'Alice';
  name = 42;

  final age = 30;
  age = 31;

  dynamic score = 100;
  score = 'excellent';
  print(score.toUpperCase());
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  // Ошибка 1: var name выводит тип String — нельзя присвоить int
  String name = 'Alice'; // или: dynamic / Object
  print(name);

  // Ошибка 2: final нельзя переприсваивать
  int age = 30;
  print(age);

  // dynamic — компилируется, toUpperCase() проверяется в рантайме
  dynamic score = 100;
  score = 'excellent';
  print(score.toUpperCase());
}
```

Ошибка 1: `var name = 'Alice'` закрепляет тип `String`, присвоение `42` (`int`) — ошибка компиляции.
Ошибка 2: `final` запрещает переприсвоение — `age = 31` не скомпилируется.

</details>
