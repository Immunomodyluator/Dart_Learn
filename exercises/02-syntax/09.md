# Задание 09 (2.5): Null Safety — 🖊 Написать код

**Тема:** 02. Основы синтаксиса | **Подтема:** 2.5 Null Safety

---

Напиши функцию `fullName(String firstName, String? lastName)`, которая возвращает полное имя. Если `lastName == null` — вернуть только `firstName`.

<details>
<summary>Решение</summary>

```dart
String fullName(String firstName, String? lastName) {
  return lastName != null ? '$firstName $lastName' : firstName;
}

void main() {
  print(fullName('Alice', 'Smith')); // Alice Smith
  print(fullName('Bob', null));      // Bob
}
```

`String?` — nullable тип. Проверка `!= null` сужает тип до `String` внутри ветки. Альтернатива: `'$firstName${lastName != null ? ' $lastName' : ''}'` или оператор `??`: `return '$firstName ${lastName ?? ''}'.trim()`.

</details>
