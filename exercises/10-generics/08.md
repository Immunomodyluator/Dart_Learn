# Задание 08 (10.4): Обобщённые функции — 🐛 Найти баг

**Тема:** 10. Дженерики | **Подтема:** 10.4 Обобщённые функции

---

Обобщённые функции выводят неверные типы или не компилируются. Найди и исправь все ошибки.

```dart
// Хотим: identity возвращает тот же тип что принял
dynamic identity(dynamic x) => x;   // теряем информацию о типе

// Хотим: мемоизация для любой функции
Map<dynamic, dynamic> memo = {};

dynamic memoize(Function fn, dynamic arg) {
  return memo[arg] ??= fn(arg);
}

// Хотим: безопасное приведение с fallback
T castOr<T>(Object? value, T fallback) {
  return value is T ? value : fallback;
}

void main() {
  final s = identity('hello');
  print(s.length);     // dynamic — нет подсказки типа

  final cached = memoize((int x) => x * x, 5);
  final cached2 = memoize((int x) => x * x, 5);
  print(cached);       // 25
  print(identical(cached, cached2)); // должно быть true

  final n = castOr<int>('not int', 0);
  print(n); // 0
}
```

<summary>Решение</summary>

```dart
// 1. identity — используем generic вместо dynamic
T identity<T>(T x) => x;

// 2. Типобезопасная мемоизация с generic
class Memoizer<A, R> {
  final Map<A, R> _cache = {};
  final R Function(A) _fn;

  Memoizer(this._fn);

  R call(A arg) => _cache.putIfAbsent(arg, () => _fn(arg));
}

// 3. castOr — уже корректен, оставляем без изменений
T castOr<T>(Object? value, T fallback) {
  return value is T ? value : fallback;
}

void main() {
  final s = identity('hello');
  print(s.length);      // 5 — тип String выведен

  final squareMemo = Memoizer<int, int>((x) => x * x);
  final c1 = squareMemo(5); // 25
  final c2 = squareMemo(5); // из кэша
  print(c1);                // 25
  print(c1 == c2);          // true

  final n = castOr<int>('not int', 0);
  print(n); // 0
}
```

**Два бага:** 1) `dynamic identity(dynamic x)` — теряет тип; generic `T identity<T>(T x)` сохраняет тип, компилятор может проверить `.length` на `String`. 2) `memoize(Function, dynamic)` — один глобальный кэш для всех функций: коллизии ключей между разными функциями с одним аргументом. Решение — отдельный `Memoizer<A, R>` объект для каждой функции.
