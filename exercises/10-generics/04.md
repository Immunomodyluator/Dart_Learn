# Задание 04 (10.2): Ограничения типов (bounds) — 🐛 Найти баг

**Тема:** 10. Дженерики | **Подтема:** 10.2 Ограничения типов (bounds)

---

Функции используют типы без необходимых ограничений. Найди и исправь все ошибки.

```dart
// sort без bound — нет compareTo
List<T> sortedCopy<T>(List<T> list) {
  final copy = List<T>.from(list);
  copy.sort();   // sort без компаратора требует Comparable
  return copy;
}

// Попытка вызвать метод без bound
T doubled<T>(T value) => value * 2;  // * не определён для произвольного T

// Неверный bound
T clamp<T extends num>(T value, num min, num max) {
  if (value < min) return min;   // num, не T
  if (value > max) return max;   // num, не T
  return value;
}

void main() {
  print(sortedCopy([3, 1, 2]));       // [1, 2, 3]
  print(doubled(5));                  // 10
  print(clamp(15, 0, 10));            // 10
}
```

<summary>Решение</summary>

```dart
// Добавить bound Comparable<T>
List<T> sortedCopy<T extends Comparable<T>>(List<T> list) {
  final copy = List<T>.from(list);
  copy.sort();
  return copy;
}

// Ограничить num для арифметики
num doubled(num value) => value * 2;
// Или через extension, но проще специализировать:
// int doubledInt(int v) => v * 2;

// clamp: возвращать T, не num
T clamp<T extends num>(T value, T lo, T hi) {
  if (value < lo) return lo;
  if (value > hi) return hi;
  return value;
}

void main() {
  print(sortedCopy([3, 1, 2]));          // [1, 2, 3]
  print(sortedCopy(['b', 'a', 'c']));    // [a, b, c]
  print(doubled(5));                     // 10
  print(doubled(3.5));                   // 7.0
  print(clamp(15, 0, 10));              // 10
  print(clamp(5.5, 0.0, 4.0));          // 4.0
}
```

**Три бага:** 1) `sort()` без компаратора требует элементы быть `Comparable` — добавлен bound `T extends Comparable<T>`. 2) `value * 2` не определён для произвольного `T` — ограничить `num`. 3) `clamp` возвращал `num` вместо `T` — параметры и возврат исправлены на `T`.
