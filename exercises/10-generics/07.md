# Задание 07 (10.4): Обобщённые функции — 🖊 Написать код

**Тема:** 10. Дженерики | **Подтема:** 10.4 Обобщённые функции

---

Напиши библиотеку утилит с обобщёнными функциями:

1. `Option<T>` — sealed-класс с `Some<T>` и `None<T>`. Функция `Option<T> tryGet<T>(List<T> list, int index)` — `Some` если индекс в диапазоне, иначе `None`.
2. `R Function(T) compose<T, R, M>(R Function(M) f, M Function(T) g)` — композиция двух функций.

<details>
<summary>Решение</summary>

```dart
// --- Option ---
sealed class Option<T> {}
final class Some<T> extends Option<T> {
  final T value;
  const Some(this.value);
  @override String toString() => 'Some($value)';
}
final class None<T> extends Option<T> {
  const None();
  @override String toString() => 'None';
}

Option<T> tryGet<T>(List<T> list, int index) {
  if (index < 0 || index >= list.length) return None<T>();
  return Some(list[index]);
}

// --- Composition ---
R Function(T) compose<T, M, R>(R Function(M) f, M Function(T) g) {
  return (T x) => f(g(x));
}

void main() {
  final nums = [10, 20, 30];
  print(tryGet(nums, 1));  // Some(20)
  print(tryGet(nums, 5));  // None

  switch (tryGet(nums, 0)) {
    case Some(:final value): print('Got: $value'); // Got: 10
    case None(): print('Missing');
  }

  // compose: String → int → double
  final parseAndHalf = compose<String, int, double>(
    (n) => n / 2,
    int.parse,
  );
  print(parseAndHalf('42')); // 21.0
}
```

Обобщённые функции (function generics) задают параметры типа непосредственно у функции `<T, M, R>`. Компилятор выводит их при вызове или принимает явно. `compose` типобезопасно связывает функции: тип выхода `g` должен совпадать с типом входа `f`.
