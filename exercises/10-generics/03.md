# Задание 03 (10.2): Ограничения типов (bounds) — 🖊 Написать код

**Тема:** 10. Дженерики | **Подтема:** 10.2 Ограничения типов (bounds)

Напиши обобщённые функции с ограничениями:

1. `T max<T extends Comparable<T>>(T a, T b)` — возвращает большее из двух значений.
2. `List<T> topN<T extends Comparable<T>>(List<T> items, int n)` — возвращает `n` наибольших элементов в порядке убывания.
3. `Map<K, List<V>> groupBy<T, K, V>(List<T> items, K Function(T) key, V Function(T) value)` — группирует по ключу.

<details>
<summary>Решение</summary>

```dart
T max<T extends Comparable<T>>(T a, T b) => a.compareTo(b) >= 0 ? a : b;

List<T> topN<T extends Comparable<T>>(List<T> items, int n) {
  final sorted = List<T>.from(items)..sort((a, b) => b.compareTo(a));
  return sorted.take(n).toList();
}

Map<K, List<V>> groupBy<T, K, V>(
  List<T> items,
  K Function(T) key,
  V Function(T) value,
) {
  final result = <K, List<V>>{};
  for (final item in items) {
    result.putIfAbsent(key(item), () => []).add(value(item));
  }
  return result;
}

void main() {
  print(max(3, 7));                 // 7
  print(max('apple', 'banana'));    // banana

  print(topN([3, 1, 4, 1, 5, 9, 2, 6], 3)); // [9, 6, 5]

  final words = ['apple', 'ant', 'banana', 'bear', 'cherry'];
  final grouped = groupBy(words, (w) => w[0], (w) => w);
  print(grouped);
  // {a: [apple, ant], b: [banana, bear], c: [cherry]}
}
```

`T extends Comparable<T>` — upper bound: принимаются только типы, реализующие `Comparable<T>`. Это даёт доступ к методу `compareTo`. Несколько параметров типа `<T, K, V>` позволяют описать сложные преобразования типобезопасно.
