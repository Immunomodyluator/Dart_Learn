# Задание 01 (10.1): Обобщённые классы — 🖊 Написать код

**Тема:** 10. Дженерики | **Подтема:** 10.1 Обобщённые классы

---

Реализуй обобщённый стек `Stack<T>`:

- Внутреннее хранилище — `List<T>`.
- `void push(T item)` — добавить в вершину.
- `T pop()` — извлечь с вершины; бросать `StateError('Stack is empty')` если пусто.
- `T get peek` — просмотр вершины без удаления.
- `bool get isEmpty`.
- `int get size`.

<details>
<summary>Решение</summary>

```dart
class Stack<T> {
  final List<T> _items = [];

  void push(T item) => _items.add(item);

  T pop() {
    if (isEmpty) throw StateError('Stack is empty');
    return _items.removeLast();
  }

  T get peek {
    if (isEmpty) throw StateError('Stack is empty');
    return _items.last;
  }

  bool get isEmpty => _items.isEmpty;
  int  get size    => _items.length;

  @override
  String toString() => 'Stack$_items';
}

void main() {
  final s = Stack<int>();
  s.push(1);
  s.push(2);
  s.push(3);
  print(s);           // Stack[1, 2, 3]
  print(s.peek);      // 3
  print(s.pop());     // 3
  print(s.size);      // 2

  final words = Stack<String>();
  words.push('hello');
  words.push('dart');
  print(words.pop()); // dart
}
```

`Stack<T>` — параметр типа `T` задаётся при использовании класса. Компилятор проверяет типы статически: `Stack<int>.push('text')` — ошибка. `removeLast()` удаляет и возвращает последний элемент — O(1) для `List`.
