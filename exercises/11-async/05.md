# Задание 05 (11.3): StreamController — 🖊 Написать код

**Тема:** 11. Асинхронность | **Подтема:** 11.3 StreamController

---

Реализуй простой event bus с помощью `StreamController`:

- Класс `EventBus` с методами:
  - `void emit<T>(T event)` — отправить событие.
  - `Stream<T> on<T>()` — подписаться только на события типа `T`.
  - `void dispose()` — закрыть контроллер.

<details>
<summary>Решение</summary>

```dart
import 'dart:async';

class EventBus {
  final StreamController _controller = StreamController.broadcast();

  void emit<T>(T event) {
    if (_controller.isClosed) return;
    _controller.add(event);
  }

  Stream<T> on<T>() => _controller.stream.whereType<T>();

  void dispose() => _controller.close();
}

// --- Использование ---
void main() async {
  final bus = EventBus();

  // Подписаться на строки
  final strSub = bus.on<String>().listen((s) => print('String: $s'));

  // Подписаться на числа
  final intSub = bus.on<int>().listen((n) => print('Int: $n'));

  bus.emit('hello');    // String: hello
  bus.emit(42);         // Int: 42
  bus.emit(3.14);       // (никто не слушает double)
  bus.emit('world');    // String: world

  await Future.delayed(Duration.zero); // дать обработчикам отработать

  await strSub.cancel();
  await intSub.cancel();
  bus.dispose();
}
```

`StreamController.broadcast()` — контроллер, позволяющий множество подписчиков. `.add(event)` — добавить событие. `.stream.whereType<T>()` — фильтрует только события нужного типа. `dispose()` закрывает контроллер — после этого `.add` не будет иметь эффекта.
