# Задание 01 (11.1): Future и async/await — 🖊 Написать код

**Тема:** 11. Асинхронность | **Подтема:** 11.1 Future и async/await

---

Напиши асинхронный сервис получения пользователей:

1. `Future<String> fetchUser(int id)` — симулирует задержку `200ms`, возвращает `'User #$id'`.
2. `Future<void> fetchMultiple(List<int> ids)` — запрашивает всех **параллельно** через `Future.wait`, печатает результаты.
3. `Future<void> fetchSequential(List<int> ids)` — запрашивает **последовательно** через `await` в цикле, печатает каждый результат.

<details>
<summary>Решение</summary>

```dart
import 'dart:async';

Future<String> fetchUser(int id) async {
  await Future.delayed(const Duration(milliseconds: 200));
  return 'User #$id';
}

Future<void> fetchMultiple(List<int> ids) async {
  final futures = ids.map(fetchUser);
  final results = await Future.wait(futures);
  print('Parallel: $results');
}

Future<void> fetchSequential(List<int> ids) async {
  for (final id in ids) {
    final user = await fetchUser(id);
    print('Sequential: $user');
  }
}

void main() async {
  await fetchMultiple([1, 2, 3]);
  // Parallel: [User #1, User #2, User #3]  — за ~200ms

  await fetchSequential([4, 5]);
  // Sequential: User #4  — за ~200ms
  // Sequential: User #5  — за ~200ms (итого ~400ms)
}
```

`Future.wait(iterable)` выполняет все Future **параллельно** и ждёт завершения всех — общее время ≈ максимуму. `await` в цикле — **последовательно**: каждый следующий запрос начинается после завершения предыдущего, время суммируется. `main() async` — точка входа может быть `async`.

</details>
