# Задание 10 (11.5): async generators — 🐛 Найти баг

**Тема:** 11. Асинхронность | **Подтема:** 11.5 async* / sync* генераторы

---

Генераторы объявлены с неверными ключевыми словами. Найди и исправь все ошибки.

```dart
// Должен возвращать Iterable<int> (sync*), но объявлен как async*
Stream<int> fibonacci() async* {
  var a = 0, b = 1;
  while (true) {
    yield a;
    final next = a + b;
    a = b;
    b = next;
  }
}

// Должен возвращать Stream<String> (async*), объявлен как sync*
Iterable<String> events() sync* {
  yield 'start';
  await Future.delayed(Duration(seconds: 1)); // await в sync* — запрещено
  yield 'end';
}

void main() {
  print(fibonacci().take(8).toList()); // [0, 1, 1, 2, 3, 5, 8, 13]
}
```

<details>
<summary>Решение</summary>

```dart
import 'dart:async';

// sync* — возвращает Iterable, не Stream
Iterable<int> fibonacci() sync* {
  var a = 0, b = 1;
  while (true) {
    yield a;
    final next = a + b;
    a = b;
    b = next;
  }
}

// async* — возвращает Stream, позволяет await
Stream<String> events() async* {
  yield 'start';
  await Future.delayed(const Duration(seconds: 1));
  yield 'end';
}

Future<void> main() async {
  print(fibonacci().take(8).toList());
  // [0, 1, 1, 2, 3, 5, 8, 13]

  await for (final e in events()) {
    print(e);
  }
  // start  (пауза 1с)  end
}
```

**Два бага:** 1) `fibonacci` объявлен как `async*` → возвращает `Stream`, не `Iterable`; нужен `sync*`. 2) `events` объявлен как `sync*` с `await` — `await` запрещён в `sync*`-генераторе (синтаксическая ошибка); нужен `async*` и тип возврата `Stream<String>`.

</details>
