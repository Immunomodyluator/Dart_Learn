# Задание 03 (11.2): Stream — 🖊 Написать код

**Тема:** 11. Асинхронность | **Подтема:** 11.2 Stream

---

Напиши функции для работы со Stream:

1. `Stream<int> counter(int from, int to, Duration interval)` — генерирует числа от `from` до `to` с заданным интервалом через `async*` / `yield`.
2. Подпишись на `counter(1, 5, Duration(milliseconds: 100))` и напечатай каждое значение. Отмени подписку после первых 3 значений.

<details>
<summary>Решение</summary>

```dart
import 'dart:async';

Stream<int> counter(int from, int to, Duration interval) async* {
  for (var i = from; i <= to; i++) {
    await Future.delayed(interval);
    yield i;
  }
}

Future<void> main() async {
  final stream = counter(1, 5, const Duration(milliseconds: 100));

  var count = 0;
  late StreamSubscription<int> sub;

  sub = stream.listen(
    (value) {
      print('Value: $value');
      count++;
      if (count >= 3) {
        sub.cancel();
        print('Cancelled after 3 values');
      }
    },
    onDone: () => print('Stream done'),
    onError: (e) => print('Error: $e'),
  );

  // Подождать завершения
  await Future.delayed(const Duration(milliseconds: 500));
}
// Value: 1
// Value: 2
// Value: 3
// Cancelled after 3 values
```

`async*` — генераторная функция, возвращающая `Stream`. `yield` отправляет значение подписчику. `StreamSubscription.cancel()` прекращает получение событий. `late` позволяет использовать `sub` внутри `listen`-коллбека до инициализации переменной.

</details>
