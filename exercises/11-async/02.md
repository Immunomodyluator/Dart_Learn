# Задание 02 (11.1): Future и async/await — 🐛 Найти баг

**Тема:** 11. Асинхронность | **Подтема:** 11.1 Future и async/await

---

Код запускается без ошибок, но результаты не выводятся. Найди и исправь все ошибки.

```dart
import 'dart:async';

Future<int> compute(int n) async {
  await Future.delayed(Duration(milliseconds: 100));
  return n * n;
}

void processResults() async {
  final result = compute(5);   // не await — result это Future<int>
  print('Result: $result');    // печатает "Instance of 'Future<int>'"

  // Fire-and-forget без обработки ошибок
  Future.delayed(Duration(seconds: 1), () {
    throw Exception('Background error'); // уходит в unhandled
  });
}

void main() {
  processResults();   // main не ждёт завершения
  print('Done');      // печатается раньше result
}
```

<details>
<summary>Решение</summary>

```dart
import 'dart:async';

Future<int> compute(int n) async {
  await Future.delayed(const Duration(milliseconds: 100));
  return n * n;
}

Future<void> processResults() async {
  final result = await compute(5);   // добавлен await
  print('Result: $result');          // Result: 25

  // Обработка ошибок fire-and-forget через .catchError или try/catch
  Future.delayed(const Duration(seconds: 1), () {
    throw Exception('Background error');
  }).catchError((e) => print('Caught background: $e'));
}

Future<void> main() async {   // main ждёт завершения
  await processResults();
  print('Done');
}
```

**Три бага:** 1) `final result = compute(5)` без `await` — `result` имеет тип `Future<int>`, а не `int`; печатается строковое представление Future. 2) `void main()` без `async`/`await` — программа не ждёт `processResults()` и завершается до вывода результата. 3) Необработанная ошибка в fire-and-forget Future — может привести к `UnhandledPromiseRejection`; нужен `.catchError`.

</details>
