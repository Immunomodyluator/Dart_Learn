# Задание 09 (11.5): async generators — 🖊 Написать код

**Тема:** 11. Асинхронность | **Подтема:** 11.5 async* / sync* генераторы

---

Напиши генераторы обоих видов:

1. `Iterable<int> range(int start, int end, [int step = 1])` — синхронный генератор через `sync*` / `yield`.
2. `Stream<String> readLines(List<String> lines, Duration delay)` — асинхронный генератор: возвращает каждую строку с задержкой `delay`.

<details>
<summary>Решение</summary>

```dart
// 1. Синхронный генератор
Iterable<int> range(int start, int end, [int step = 1]) sync* {
  assert(step > 0, 'step must be positive');
  for (var i = start; i < end; i += step) {
    yield i;
  }
}

// 2. Асинхронный генератор
Stream<String> readLines(List<String> lines, Duration delay) async* {
  for (final line in lines) {
    await Future.delayed(delay);
    yield line;
  }
}

Future<void> main() async {
  // sync* — ленивый Iterable
  print(range(0, 10, 2).toList()); // [0, 2, 4, 6, 8]
  print(range(1, 6).toList());     // [1, 2, 3, 4, 5]

  // async* — Stream
  final lines = ['Line 1', 'Line 2', 'Line 3'];
  await for (final line in readLines(lines, Duration(milliseconds: 50))) {
    print(line);
  }
  // Line 1
  // Line 2
  // Line 3
}
```

`sync*` / `yield` — ленивый синхронный итератор: значения вычисляются по мере потребления. `async*` / `yield` — возвращает `Stream`; `await` внутри корректно работает. `yield*` делегирует другому итерируемому/потоку. Генераторы приостанавливаются при `yield` и возобновляются при следующем запросе.
