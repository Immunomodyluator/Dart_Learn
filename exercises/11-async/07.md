# Задание 07 (11.4): Isolate — 🖊 Написать код

**Тема:** 11. Асинхронность | **Подтема:** 11.4 Isolate

---

Напиши программу, которая выполняет тяжёлое вычисление в отдельном изоляте:

1. Функция `int primeCount(int limit)` — считает количество простых чисел до `limit` (решето Эратосфена).
2. Запусти `primeCount(1000000)` в отдельном изоляте через `Isolate.run` (Dart 2.19+).
3. Измерь время выполнения с изолятом и без.

<details>
<summary>Решение</summary>

```dart
import 'dart:isolate';

int primeCount(int limit) {
  if (limit < 2) return 0;
  final sieve = List<bool>.filled(limit + 1, true);
  sieve[0] = sieve[1] = false;
  for (var i = 2; i * i <= limit; i++) {
    if (sieve[i]) {
      for (var j = i * i; j <= limit; j += i) {
        sieve[j] = false;
      }
    }
  }
  return sieve.where((b) => b).length;
}

Future<void> main() async {
  const limit = 1000000;

  // Без изолята (блокирует event loop)
  final t1 = DateTime.now();
  final count1 = primeCount(limit);
  final ms1 = DateTime.now().difference(t1).inMilliseconds;
  print('Main isolate: $count1 primes in ${ms1}ms');

  // В отдельном изоляте (не блокирует event loop)
  final t2 = DateTime.now();
  final count2 = await Isolate.run(() => primeCount(limit));
  final ms2 = DateTime.now().difference(t2).inMilliseconds;
  print('Spawned isolate: $count2 primes in ${ms2}ms');
  // 78498 primes — результаты совпадают
}
```

`Isolate.run(fn)` — запускает функцию в новом изоляте и возвращает `Future<R>`. Изоляты не разделяют память — вся коммуникация через сообщения. Для длительных вычислений (> ~1ms) изолят предотвращает блокировку event loop и зависание UI.
