<summary>Решение</summary>

```bash
# (A) Установить webdev глобально (или добавить в dev_dependencies)
dart pub global activate webdev

# (B) Dev-сервер с hot reload
dart run webdev serve
# или: webdev serve (если PATH настроен)

# (C) Продакшн-билд (минификация + tree shaking)
dart run webdev build --output web:build

# (D) Source maps (включены по умолчанию в serve, для build явно):
dart run webdev serve --enable-experimental-hot-reload
# Source maps включены автоматически в dev-режиме
```

```yaml
# pubspec.yaml
name: my_web_app
environment:
  sdk: ^3.3.0 # (E) ключ 'environment'

dependencies:
  web: ^0.3.0 # (F) пакет 'web' — современный dart:js_interop API

dev_dependencies:
  build_runner: ^2.4.0
  build_web_compilers: ^3.0.0 # (G) собирает и транспилирует через dart2js/DDC

targets:
  $default:
    builders:
      build_web_compilers|entrypoint:
        options:
          compiler: dart2js # (H) dart2js для prod, ddc для dev
          # dart2js_args:
          #   - -O2           # уровень оптимизации
```

```dart
// web/main.dart
import 'dart:html'; // (I) — dart:html для DOM API (устаревший, но широко используемый)
// Современная альтернатива: package:web (dart:js_interop)

void main() {
  // (J) querySelector — ищет элемент по CSS-селектору
  final button = querySelector('#myButton') as ButtonElement;  // (J)

  // (K) addEventListener
  button.addEventListener('click', (event) {                   // (K)
    final input = querySelector('#nameInput') as InputElement; // (J второй)
    final name = input.value ?? '';                            // (L) .value
    querySelector('#result')!.text = 'Hello, $name!';         // (M)
  });
}
```

**Ключевые команды и файлы:** `webdev serve` — dev-сервер с DDC (быстрая инкрементная компиляция); `webdev build` — prod-сборка через dart2js с tree shaking и минификацией. `build_web_compilers` управляет выбором компилятора через `build.yaml`. `dart:html` — классический DOM API; `package:web` — современный нулевой накладной расход на основе `dart:js_interop`.
