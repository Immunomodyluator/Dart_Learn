Код падает с `Concurrent modification during iteration`. Найди и исправь ошибку.

```dart
void removeZeroScores(Map<String, int> scores) {
  for (final key in scores.keys) {
    if (scores[key] == 0) {
      scores.remove(key);  // удаление во время итерации
    }
  }
}

void main() {
  final scores = {'Alice': 90, 'Bob': 0, 'Carol': 0, 'Dave': 55};
  removeZeroScores(scores);
  print(scores); // ожидается {Alice: 90, Dave: 55}
}
```

<summary>Решение</summary>

```dart
void removeZeroScores(Map<String, int> scores) {
  // Собрать ключи заранее, потом удалить
  final toRemove = scores.keys.where((k) => scores[k] == 0).toList();
  for (final key in toRemove) {
    scores.remove(key);
  }
}

void main() {
  final scores = {'Alice': 90, 'Bob': 0, 'Carol': 0, 'Dave': 55};
  removeZeroScores(scores);
  print(scores); // {Alice: 90, Dave: 55}
}
```

Нельзя изменять коллекцию (добавлять/удалять) прямо во время итерации по ней — Dart бросает `ConcurrentModificationError`. Решение: сначала собрать ключи для удаления в отдельный список (`.toList()` материализует итерацию), затем удалять из оригинальной Map.
