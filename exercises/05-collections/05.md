Дополни пропуски согласно комментариям.

```dart
void main() {
  final scores = <String, int>{'Alice': 90, 'Bob': 75};

  // Получить значение или 0, если ключ отсутствует
  final carol = scores___ ?? ___;

  // Добавить ключ только если его нет, иначе оставить старое
  scores.___('Diana', () => 85);

  // Обновить существующее значение: прибавить 5
  scores.___('Alice', (old) => old + 5);

  // Итерация по парам
  scores.___((___, ___) {
    print('$key: $value');
  });
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  final scores = <String, int>{'Alice': 90, 'Bob': 75};

  final carol = scores['Carol'] ?? 0;

  scores.putIfAbsent('Diana', () => 85);

  scores.update('Alice', (old) => old + 5);

  scores.forEach((key, value) {
    print('$key: $value');
  });
}
```

`map[key] ?? default` — безопасное чтение. `putIfAbsent` — добавляет запись, только если ключа нет. `update` — изменяет существующее значение через функцию (бросает `ArgumentError`, если ключ не найден; добавить параметр `ifAbsent` для вставки нового).
