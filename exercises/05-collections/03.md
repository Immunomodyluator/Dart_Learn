# Задание 03 (5.2): Set — множества — 🖊 Написать код

**Тема:** 05. Коллекции | **Подтема:** 5.2 Set — множества

Напиши функцию `uniqueOrdered<T>(List<T> list)`, которая возвращает список уникальных элементов, сохраняя исходный порядок первого появления.

Требования:

- Используй `LinkedHashSet` из `dart:collection`.
- Сигнатура: `List<T> uniqueOrdered<T>(List<T> list)`

Пример:

```
uniqueOrdered([3, 1, 2, 1, 3, 4]) → [3, 1, 2, 4]
uniqueOrdered(['a', 'b', 'a'])   → [a, b]
```

<details>
<summary>Решение</summary>

```dart
import 'dart:collection';

List<T> uniqueOrdered<T>(List<T> list) {
  return LinkedHashSet<T>.from(list).toList();
}

void main() {
  print(uniqueOrdered([3, 1, 2, 1, 3, 4])); // [3, 1, 2, 4]
  print(uniqueOrdered(['a', 'b', 'a']));     // [a, b]
}
```

`LinkedHashSet` — реализация `Set`, которая хранит элементы в порядке вставки. `Set.from` автоматически убирает дубли. Финальный `.toList()` возвращает нужный тип. Обычный `HashSet` не гарантирует порядок.

</details>
