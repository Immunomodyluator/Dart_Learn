# Задание 10 (5.5): Collection if / for / spread — 🐛 Найти баг

**Тема:** 05. Коллекции | **Подтема:** 5.5 Collection if / for / spread

---

Код не компилируется. Найди и исправь все ошибки.

```dart
void main() {
  final List<String>? extras = null;

  final items = [
    'always',
    ...?['opt1', 'opt2'],   // spread nullable List — ok?
    ...extras,              // extras может быть null
    if true 'debug',        // условие без скобок
    for (var i = 0; i < 3; i) 'item$i',  // нет инкремента
  ];

  print(items);
}
```

<summary>Решение</summary>

```dart
void main() {
  final List<String>? extras = null;

  final items = [
    'always',
    ...['opt1', 'opt2'],   // список не null — убрать ?
    ...?extras,            // extras nullable — нужен ...?
    if (true) 'debug',     // скобки обязательны
    for (var i = 0; i < 3; i++) 'item$i',  // i++ вместо i
  ];

  print(items);
  // [always, opt1, opt2, debug, item0, item1, item2]
}
```

**Четыре бага:** 1) `...?['opt1', 'opt2']` — литерал не nullable, `?` не нужен. 2) `...extras` — spread по nullable требует `...?`. 3) `if true` → `if (true)` — скобки обязательны. 4) `i` в `for` не инкрементируется — бесконечный цикл; нужен `i++`.
