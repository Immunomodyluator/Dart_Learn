# Задание 07 (5.4): Методы высшего порядка — 🖊 Написать код

**Тема:** 05. Коллекции | **Подтема:** 5.4 Методы высшего порядка

---

Напиши функцию `sumOfSquaresOfEvens(List<int> numbers)`, которая возвращает сумму квадратов чётных чисел из списка.

Требования:

- Использовать цепочку `.where()` → `.map()` → `.fold()`.
- Без промежуточных переменных (всё в одном выражении).

Пример:

```
sumOfSquaresOfEvens([1, 2, 3, 4, 5]) → 20   // 2²+4² = 4+16
sumOfSquaresOfEvens([1, 3, 5])       → 0
```

<details>
<summary>Решение</summary>

```dart
int sumOfSquaresOfEvens(List<int> numbers) {
  return numbers
      .where((n) => n.isEven)
      .map((n) => n * n)
      .fold(0, (sum, n) => sum + n);
}

void main() {
  print(sumOfSquaresOfEvens([1, 2, 3, 4, 5])); // 20
  print(sumOfSquaresOfEvens([1, 3, 5]));        // 0
}
```

`.where` — фильтрация (предикат). `.map` — преобразование каждого элемента. `.fold(initial, combine)` — свёртка с начальным значением. Все три возвращают ленивый `Iterable`, поэтому цепочка не создаёт лишних списков — вычисление происходит один проход.

</details>
