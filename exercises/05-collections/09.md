void buildMenu({required bool isAdmin, List<String>? extras}) {
  final base = ['Home', 'Profile'];

  final menu = [
    ___base,                      // spread базового меню
    ___ isAdmin ___ ...adminItems,// добавить adminItems только если isAdmin

  buildMenu(isAdmin: true, extras: ['news', 'help']);

<summary>Решение</summary>

```dart
void buildMenu({required bool isAdmin, List<String>? extras}) {
  final base = ['Home', 'Profile'];
  final adminItems = ['Users', 'Settings'];

  final menu = [
    ...base,
    if (isAdmin) ...adminItems,
    for (final item in extras ?? [])
      item.toUpperCase(),
  ];

  print(menu);
}

void main() {
  buildMenu(isAdmin: true, extras: ['news', 'help']);
  // [Home, Profile, Users, Settings, NEWS, HELP]
  buildMenu(isAdmin: false);
  // [Home, Profile]
}
```

`...list` — вставляет все элементы другого итерируемого. `if (cond) element` / `if (cond) ...list` — условное включение. `for (var x in iterable) expr` — встраивает результат выражения для каждого элемента. `extras ?? []` защищает от `null`.
