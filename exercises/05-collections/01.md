# Задание 01 (5.1): List — списки — 🔲 Дополнить код

**Тема:** 05. Коллекции | **Подтема:** 5.1 List — списки

---

Дополни пропуски: создай и оперируй списками согласно комментариям.

```dart
void main() {
  // Пустой изменяемый список строк
  final names = <String>___;

  // Добавить элемент
  names.___('Alice');
  names.___('Bob');

  // Вставить в начало
  names.___(0, 'Zara');

  // Удалить по значению
  names.___('Bob');

  // Неизменяемый список из существующего
  final fixed = List.___(___, ___);

  print(names);  // [Zara, Alice]
  print(fixed);  // [Zara, Alice]
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  final names = <String>[];

  names.add('Alice');
  names.add('Bob');

  names.insert(0, 'Zara');

  names.remove('Bob');

  final fixed = List.unmodifiable(names);

  print(names);  // [Zara, Alice]
  print(fixed);  // [Zara, Alice]
}
```

`<String>[]` — пустой список с явным типом. `add` — в конец, `insert(index, value)` — по позиции, `remove(value)` — первое вхождение. `List.unmodifiable` создаёт неизменяемую копию: любая мутация бросит `UnsupportedError`.

</details>
