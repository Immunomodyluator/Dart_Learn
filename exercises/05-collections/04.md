# Задание 04 (5.2): Set — множества — 🐛 Найти баг

**Тема:** 05. Коллекции | **Подтема:** 5.2 Set — множества

---

Код должен работать с множеством целых чисел, но типы выведены неверно. Найди и исправь все ошибки.

```dart
void main() {
  var ids = {};          // предполагается Set<int>
  ids.add(1);
  ids.add(2);
  ids.add(1);            // дубль — не должен добавиться

  var others = {3, 4};
  var combined = ids.union(others);

  print(ids.length);     // ожидается 2
  print(combined);       // ожидается {1, 2, 3, 4}
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  var ids = <int>{};      // {} без элементов → Map<dynamic, dynamic>; нужен <int>{}
  ids.add(1);
  ids.add(2);
  ids.add(1);

  var others = <int>{3, 4};
  var combined = ids.union(others);

  print(ids.length);   // 2
  print(combined);     // {1, 2, 3, 4}
}
```

**Баг:** `var ids = {}` — Dart выводит тип `Map<dynamic, dynamic>`, а не `Set`. У `Map` нет метода `add`, поэтому программа падает на первой же операции. Правильная форма пустого Set: `<int>{}` или `Set<int>()`.

</details>
