# Задание 02 (5.1): List — списки — 🐛 Найти баг

**Тема:** 05. Коллекции | **Подтема:** 5.1 List — списки

---

Код падает с `UnsupportedError: Cannot add to an unmodifiable list`. Найди и исправь все ошибки.

```dart
void main() {
  const numbers = [1, 2, 3];
  numbers.add(4);
  numbers[0] = 10;

  final doubled = numbers.map((n) => n * 2);
  print(doubled.length);  // вызывается дважды — снова вычислится
  print(doubled.length);

  final sorted = numbers.sort();
  print(sorted);
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  // const-список неизменяем — использовать final или var
  final numbers = [1, 2, 3];
  numbers.add(4);
  numbers[0] = 10;

  // .map() возвращает ленивый Iterable — материализовать в List
  final doubled = numbers.map((n) => n * 2).toList();
  print(doubled.length);
  print(doubled.length);

  // .sort() мутирует список и возвращает void
  numbers.sort();
  print(numbers);
}
```

**Три бага:** 1) `const [...]` — неизменяемый, нужен `final`. 2) `.map()` возвращает ленивый `MappedIterable` — без `.toList()` каждый `length` пересчитывает заново. 3) `.sort()` сортирует «на месте» и возвращает `void` — результат нельзя связать с переменной.
