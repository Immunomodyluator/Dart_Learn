# Задание 01 (12.1): Пользовательские исключения — 🖊 Написать код

**Тема:** 12. Обработка ошибок | **Подтема:** 12.1 Пользовательские исключения

---

Создай иерархию пользовательских исключений для сервиса аутентификации:

- `abstract class AuthException implements Exception` — поля `String message`, `String code`.
- `class InvalidCredentialsException extends AuthException`
- `class AccountLockedException extends AuthException` — дополнительное поле `DateTime lockedUntil`.
- `class TokenExpiredException extends AuthException`

Напиши функцию `String authenticate(String login, String password)`, которая симулирует проверку и бросает нужные исключения.

<details>
<summary>Решение</summary>

```dart
abstract class AuthException implements Exception {
  final String message;
  final String code;
  const AuthException(this.message, this.code);

  @override
  String toString() => 'AuthException[$code]: $message';
}

class InvalidCredentialsException extends AuthException {
  const InvalidCredentialsException()
      : super('Invalid login or password', 'AUTH_001');
}

class AccountLockedException extends AuthException {
  final DateTime lockedUntil;
  AccountLockedException(this.lockedUntil)
      : super('Account locked until $lockedUntil', 'AUTH_002');
}

class TokenExpiredException extends AuthException {
  const TokenExpiredException()
      : super('Session token has expired', 'AUTH_003');
}

String authenticate(String login, String password) {
  if (login == 'locked') {
    throw AccountLockedException(
      DateTime.now().add(const Duration(minutes: 30)),
    );
  }
  if (password != 'secret') throw const InvalidCredentialsException();
  return 'token_${login}_${DateTime.now().millisecondsSinceEpoch}';
}

void main() {
  for (final creds in [('admin', 'secret'), ('user', 'wrong'), ('locked', 'x')]) {
    try {
      final token = authenticate(creds.$1, creds.$2);
      print('OK: $token');
    } on AccountLockedException catch (e) {
      print('Locked: ${e.message}');
    } on AuthException catch (e) {
      print('Auth error [${e.code}]: ${e.message}');
    }
  }
}
```

`implements Exception` — соответствует интерфейсу `Exception` без наследования реализации. Иерархия исключений позволяет перехватывать как конкретный тип, так и весь базовый класс. `const AuthException(...)` — позволяет создавать compile-time константы.
