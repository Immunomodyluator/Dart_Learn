# Задание 05 (12.3): Zone и обработка необработанных ошибок — 🖊 Написать код

**Тема:** 12. Обработка ошибок | **Подтема:** 12.3 Zone и обработка необработанных ошибок

---

Реализуй систему глобального перехвата ошибок с помощью зон Dart.

**Задача:**

1. Напиши функцию `runApp(Future<void> Function() body)`, которая запускает `body` через `runZonedGuarded`. Обработчик ошибок зоны должен:
   - Печатать время ошибки, тип и сообщение в формате: `[2024-01-01 12:00:00] UNHANDLED ERROR: ...`
   - Записывать ошибки в список `List<String> errorLog`

2. Напиши `Future<void> riskyOperation()`, которая:
   - Запускает «fire-and-forget» Future (без `await`) который бросает исключение через 20 мс
   - Запускает Timer через 40 мс, который тоже бросает исключение
   - Возвращает управление сразу (не ждёт)

3. В `main` вызови `runApp(() => riskyOperation())`, подожди 100 мс, распечатай `errorLog`.

```dart
import 'dart:async';

final List<String> errorLog = [];

// TODO: реализуй runApp, riskyOperation и main
```

<details>
<summary>Решение</summary>

```dart
import 'dart:async';

final List<String> errorLog = [];

void runApp(Future<void> Function() body) {
  runZonedGuarded(
    body,
    (error, stackTrace) {
      final timestamp = DateTime.now().toString().substring(0, 19);
      final msg = '[$timestamp] UNHANDLED ERROR: ${error.runtimeType}: $error';
      print(msg);
      errorLog.add(msg);
    },
  );
}

Future<void> riskyOperation() async {
  // fire-and-forget — ошибка будет поймана зоной
  Future.delayed(const Duration(milliseconds: 20), () {
    throw StateError('async fire-and-forget error');
  });

  // Timer тоже управляется зоной
  Timer(const Duration(milliseconds: 40), () {
    throw Exception('timer callback error');
  });
}

void main() async {
  runApp(() => riskyOperation());
  await Future.delayed(const Duration(milliseconds: 100));

  print('\n--- Error Log (${errorLog.length} entries) ---');
  for (final entry in errorLog) {
    print(entry);
  }
}
// [2024-...] UNHANDLED ERROR: StateError: Bad state: async fire-and-forget error
// [2024-...] UNHANDLED ERROR: _Exception: Exception: timer callback error
// --- Error Log (2 entries) ---
```

`runZonedGuarded` создаёт зону, которая перехватывает все **необработанные** ошибки — в том числе из fire-and-forget Future и коллбеков Timer. Обычный `try/catch` не поможет для кода, запущенного без `await`. Зоны — стандартный механизм для глобальной обработки ошибок во Flutter и Dart-серверах.
