Код должен отсортировать объекты по приоритету. Найди все баги.

```dart
import 'dart:collection';

class Task implements Comparable {   // bug 1 — не указан type parameter
  final String name;
  final int priority;

  Task(this.name, this.priority);

  @override
  int compareTo(other) {             // bug 2 — правильно ascending, но мы хотим DESC
    return priority.compareTo(other.priority);
  }
}

void main() {
  final queue = HeapPriorityQueue<Task>();  // из dart:collection
  queue.add(Task('low', 1));
  queue.add(Task('high', 10));
  queue.add(Task('medium', 5));

  while (queue.isNotEmpty) {
    final task = queue.removeFirst();
    print('${task.name}: ${task.priority}');
  }
  // ожидаем: high, medium, low (но выведет low, medium, high) — bug 2

  // попытка сравнить задачи напрямую
  final a = Task('a', 3);
  final b = Task('b', 7);
  if (a > b) {                       // bug 3 — оператор > не определён для Task
    print('a wins');
  }
}
```

<summary>Решение</summary>

```dart
import 'dart:collection';

class Task implements Comparable<Task> {   // fix 1: type parameter
  final String name;
  final int priority;

  Task(this.name, this.priority);

  @override
  int compareTo(Task other) {        // fix 2: DESC — инвертируем порядок
    return other.priority.compareTo(priority);
  }

  // fix 3: оператор > через compareTo
  bool operator >(Task other) => compareTo(other) < 0;
  bool operator <(Task other) => compareTo(other) > 0;

  @override
  String toString() => '$name($priority)';
}

void main() {
  final queue = HeapPriorityQueue<Task>();
  queue.add(Task('low', 1));
  queue.add(Task('high', 10));
  queue.add(Task('medium', 5));

  while (queue.isNotEmpty) {
    final task = queue.removeFirst();
    print('${task.name}: ${task.priority}');
  }
  // high: 10
  // medium: 5
  // low: 1

  final a = Task('a', 3);
  final b = Task('b', 7);
  if (a > b) {
    print('a wins');
  } else {
    print('b wins'); // b wins — b имеет больший приоритет
  }
}
```

**Три бага:** 1) `Comparable` без параметра типа — `other` будет `dynamic`, теряется типобезопасность. 2) `priority.compareTo(other.priority)` возвращает ascending (маленький = первый), а `HeapPriorityQueue` использует `compareTo` для определения «наименьшего» — нужно инвертировать (`other.compareTo(this)`). 3) Операторы `>` и `<` не наследуются от `Comparable` автоматически — нужно объявить явно.
