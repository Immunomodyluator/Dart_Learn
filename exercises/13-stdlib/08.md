# Задание 08 (13.4): dart:io — 🐛 Найти баг

**Тема:** 13. Стандартная библиотека Dart | **Подтема:** 13.4 dart:io

---

Программа должна построчно читать файл и записывать результаты в другой. Найди все баги.

```dart
import 'dart:io';
import 'dart:convert';

Future<void> processFile(String inputPath, String outputPath) async {
  final input = File(inputPath);
  final output = File(outputPath);

  // bug 1 — openWrite() без await open, но это окей (sync)
  // bug 2 — IOSink не закрывается если в процессе возникнет ошибка
  final sink = output.openWrite();

  await input
      .openRead()
      .transform(utf8.decoder)
      .transform(LineSplitter())
      .forEach((line) {
        // bug 3 — нет await перед write; sink.write — не Future, но sink.writeln тоже
        // реальный баг: преобразование строк с ошибкой логики
        final processed = line.trim().toUpperCase;  // bug 4 — .toUpperCase — геттер/метод?
        sink.writeln(processed);
      });

  sink.close(); // bug 5 — close() возвращает Future, нет await
}

void main() async {
  // bug 6 — нет обработки FileSystemException если файл не существует
  await processFile('input.txt', 'output.txt');
  print('Done');
}
```

<details>
<summary>Решение</summary>

```dart
import 'dart:io';
import 'dart:convert';

Future<void> processFile(String inputPath, String outputPath) async {
  final input = File(inputPath);

  // проверяем существование входного файла
  if (!await input.exists()) {
    throw FileSystemException('Input file not found', inputPath);
  }

  final sink = File(outputPath).openWrite();
  try {                                          // fix 2: try/finally
    await input
        .openRead()
        .transform(utf8.decoder)
        .transform(const LineSplitter())         // fix: const
        .forEach((line) {
          final processed = line.trim().toUpperCase(); // fix 4: .toUpperCase() — метод!
          sink.writeln(processed);
        });
  } finally {
    await sink.flush();
    await sink.close();                          // fix 5: await close()
  }
}

void main() async {
  try {                                          // fix 6: обрабатываем FileSystemException
    await processFile('input.txt', 'output.txt');
    print('Done');
  } on FileSystemException catch (e) {
    print('File error: ${e.message} — ${e.path}');
  }
}
```

**Баги:**

1. `openWrite()` синхронный и возвращает `IOSink` — `await` не нужен, это не баг.
2. Нет `try/finally` — если исключение в `forEach`, sink останется открытым.
3. `sink.write`/`sink.writeln` — не `Future`, работают корректно без `await`.
4. `.toUpperCase` без скобок — это ссылка на **метод** (тип `String Function()`), а не вызов; нужно `.toUpperCase()`.
5. `sink.close()` без `await` — close возвращает `Future<void>`; без `await` данные могут не сбросится на диск.
6. Нет обработки `FileSystemException` в `main` — программа упадёт с необработанной ошибкой.
