**Задача:**

```dart
import 'dart:async';

Future<T> withTimeout<T>(
  Future<T> future,
  Duration timeout,
  T fallback,
) {
  final completer = Completer<T>();

  final timer = Timer(timeout, () {
    if (!completer.isCompleted) completer.complete(fallback);
  });

  future.then((value) {
    timer.cancel();
    if (!completer.isCompleted) completer.complete(value);
  }).catchError((Object error, StackTrace st) {
    timer.cancel();
    if (!completer.isCompleted) completer.completeError(error, st);
  });

  return completer.future;
}

Stream<int> ticker(Duration interval, {int count = 5}) {
  final controller = StreamController<int>();
  var i = 0;

  Timer.periodic(interval, (timer) {
    if (i >= count) {
      timer.cancel();
      controller.close();
      return;
    }
    controller.add(i++);
  });

  return controller.stream;
}

void deferredLog(String message) {
  scheduleMicrotask(() => print('[deferred] $message'));
}

void main() async {
  // 1. withTimeout
  final slow = Future.delayed(const Duration(seconds: 5), () => 'slow');
  final result = await withTimeout(slow, const Duration(milliseconds: 100), 'fallback');
  print(result); // fallback

  // 2. ticker
  deferredLog('about to start ticker'); // откладывается
  print('synchronous code runs first');
  // [deferred] about to start ticker

  await for (final tick in ticker(const Duration(milliseconds: 50))) {
    print('tick: $tick'); // 0, 1, 2, 3, 4
  }
}
```

`Completer` позволяет создать Future и завершить его вручную в нужный момент — идеально для обёрток над callback-based API. `scheduleMicrotask` откладывает выполнение до конца текущего «оборота» event loop, но **перед** следующим событием (таймером, I/O) — удобно для логирования и накопления изменений.
