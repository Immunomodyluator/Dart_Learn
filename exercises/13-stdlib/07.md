Реализуй утилиту для работы с файловой системой.

**Задача:**

1. Напиши `Future<void> writeLines(String path, List<String> lines)` — записывает строки в файл, каждую с новой строки.

2. Напиши `Future<List<String>> readNonEmptyLines(String path)` — читает файл и возвращает только непустые строки (обрезая пробелы). Если файл не существует — выбрасывай `FileSystemException`.

3. Напиши `Future<Map<String, int>> countWordsInFile(String path)` — возвращает Map «слово → количество» (без учёта регистра, слова разделяются пробелами и знаками препинания).

4. Напиши `Future<void> copyTextFile(String src, String dst)` — копирует текстовый файл построчно (через `Stream<String>` и `IOSink`).

```dart
import 'dart:io';

Future<void> writeLines(String path, List<String> lines) async {
  final file = File(path);
  await file.writeAsString(lines.join('\n'));
}

Future<List<String>> readNonEmptyLines(String path) async {
  final file = File(path);
  if (!await file.exists()) {
    throw FileSystemException('File not found', path);
  }
  final content = await file.readAsString();
  return content
      .split('\n')
      .map((l) => l.trim())
      .where((l) => l.isNotEmpty)
      .toList();
}

Future<Map<String, int>> countWordsInFile(String path) async {
  final lines = await readNonEmptyLines(path);
  final freq = <String, int>{};
  for (final line in lines) {
    for (final word in line.toLowerCase().split(RegExp(r'[\s,\.!?;:]+'))) {
      if (word.isNotEmpty) freq[word] = (freq[word] ?? 0) + 1;
    }
  }
  return freq;
}

Future<void> copyTextFile(String src, String dst) async {
  final source = File(src);
  final sink = File(dst).openWrite();
  try {
    await source
        .openRead()
        .transform(const SystemEncoding().decoder)
        .transform(const LineSplitter())
        .forEach((line) {
          sink.writeln(line);
        });
  } finally {
    await sink.flush();
    await sink.close();
  }
}

void main() async {
  const path = '/tmp/test_dart_io.txt';

  await writeLines(path, ['Hello Dart', 'dart is fast', 'Hello world']);
  final lines = await readNonEmptyLines(path);
  print(lines); // [Hello Dart, dart is fast, Hello world]

  final freq = await countWordsInFile(path);
  print(freq);  // {hello: 2, dart: 2, is: 1, fast: 1, world: 1}

  await copyTextFile(path, '/tmp/test_copy.txt');
  print(await readNonEmptyLines('/tmp/test_copy.txt'));
}
```

`File.readAsString()` читает весь файл в память — для больших файлов предпочитай потоковую обработку через `openRead()`. `IOSink` нужно явно закрывать (`close()`) — иначе данные могут не записаться на диск. При копировании через Stream обязательно используй `try/finally` для закрытия sink.
