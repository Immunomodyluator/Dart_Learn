# Задание 01 (3.1): import, export, show и hide — 🔲 Дополнить код

**Тема:** 03. Библиотеки и импорты | **Подтема:** 3.1 import, export, show и hide

---

Дополни импорты в файле, используя `show` и `hide`, чтобы:

1. Из `dart:math` импортировать **только** `sqrt` и `pi`
2. Из `dart:convert` импортировать всё, **кроме** `LineSplitter`
3. Из `package:http/http.dart` импортировать как псевдоним `http`

```dart
import 'dart:math' ___ sqrt, pi;
import 'dart:convert' ___ LineSplitter;
import 'package:http/http.dart' ___ http;

void main() {
  print(sqrt(16));   // 4.0
  print(pi);         // 3.14...
  print(jsonEncode({'key': 'value'}));
}
```

<details>
<summary>Решение</summary>

```dart
import 'dart:math' show sqrt, pi;
import 'dart:convert' hide LineSplitter;
import 'package:http/http.dart' as http;

void main() {
  print(sqrt(16));
  print(pi);
  print(jsonEncode({'key': 'value'}));
}
```

`show` — импортировать только перечисленные имена. `hide` — импортировать всё, кроме перечисленных. `as` — задаёт псевдоним для пространства имён, что удобно при конфликте имён или для читаемости (`http.get(...)` вместо просто `get`).

</details>
