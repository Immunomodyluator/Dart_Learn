# Задание 03 (3.2): Deferred imports — 🔲 Дополнить код

**Тема:** 03. Библиотеки и импорты | **Подтема:** 3.2 Deferred imports

---

Дополни код ленивой загрузки библиотеки. Заполни пропуски `___`.

```dart
import 'package:heavy_lib/heavy_lib.dart' ___ as heavy;

Future<void> main() async {
  print('App started');

  // Загрузить библиотеку только сейчас
  await heavy.___;

  // Использовать функцию из библиотеки
  heavy.doWork();
}
```

<details>
<summary>Решение</summary>

```dart
import 'package:heavy_lib/heavy_lib.dart' deferred as heavy;

Future<void> main() async {
  print('App started');

  await heavy.loadLibrary();

  heavy.doWork();
}
```

`deferred as` откладывает загрузку библиотеки до момента явного вызова `loadLibrary()`. Вызов `loadLibrary()` возвращает `Future` — нужен `await`. После загрузки все имена из библиотеки доступны через псевдоним `heavy`.

</details>
