# Задание 04 (3.2): Deferred imports — 🐛 Найти баг

**Тема:** 03. Библиотеки и импорты | **Подтема:** 3.2 Deferred imports

---

Код использует deferred import, но падает в рантайме. Найди проблему.

```dart
import 'dart:math' deferred as math;

void main() {
  print(math.sqrt(16));
}
```

<details>
<summary>Решение</summary>

```dart
import 'dart:math' deferred as math;

Future<void> main() async {
  await math.loadLibrary();
  print(math.sqrt(16));
}
```

Проблема: библиотека помечена как `deferred`, но `loadLibrary()` ни разу не вызван. Обращение к `math.sqrt` до загрузки бросает ошибку в рантайме. Нужно дождаться `await math.loadLibrary()` перед первым использованием.

</details>
