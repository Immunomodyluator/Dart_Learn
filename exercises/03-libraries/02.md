# Задание 02 (3.1): import, export, show и hide — 🐛 Найти баг

**Тема:** 03. Библиотеки и импорты | **Подтема:** 3.1 import, export, show и hide

---

Два файла должны работать вместе, но при компиляции возникает ошибка. Найди и исправь проблему.

```dart
// lib/utils.dart
String capitalize(String s) => s[0].toUpperCase() + s.substring(1);
int clamp(int val, int min, int max) => val.clamp(min, max);
```

```dart
// bin/main.dart
import '../lib/utils.dart' show capitalize;

void main() {
  print(capitalize('hello'));   // Hello
  print(clamp(15, 0, 10));      // 10
}
```

<details>
<summary>Решение</summary>

```dart
// bin/main.dart
import '../lib/utils.dart' show capitalize, clamp;

void main() {
  print(capitalize('hello'));
  print(clamp(15, 0, 10));
}
```

`show capitalize` импортирует **только** `capitalize` — имя `clamp` в этом файле недоступно. Нужно добавить `clamp` в список `show`, либо убрать `show` совсем, чтобы импортировать всё.

</details>
