# Задание 05 (3.3): part, part of и директива library — 🖊 Написать код

**Тема:** 03. Библиотеки и импорты | **Подтема:** 3.3 part, part of и директива library

---

Разбей одну библиотеку `geometry` на два файла с помощью `part` / `part of`. Класс `Circle` должен быть в `circle.dart`, класс `Rectangle` — в `rectangle.dart`. Оба файла — части главного `geometry.dart`.

Напиши заголовки всех трёх файлов (без реализации классов).

<details>
<summary>Решение</summary>

```dart
// lib/geometry.dart  — главный файл библиотеки
library geometry;

part 'circle.dart';
part 'rectangle.dart';
```

```dart
// lib/circle.dart  — часть библиотеки
part of geometry;

class Circle {
  final double radius;
  Circle(this.radius);
  double get area => 3.14159 * radius * radius;
}
```

```dart
// lib/rectangle.dart  — часть библиотеки
part of geometry;

class Rectangle {
  final double width, height;
  Rectangle(this.width, this.height);
  double get area => width * height;
}
```

`part 'file.dart'` в главном файле объявляет файл частью библиотеки. `part of library_name` — обратная ссылка. Part-файлы разделяют одно пространство имён, включая приватные (`_`) идентификаторы.

</details>
