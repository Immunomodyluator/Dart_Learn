<summary>Решение</summary>

```dart
void main() {
  final matrix = [
    [1, 2, 3],
    [4, 6, 7],
    [8, 9, 10],
  ];

  int? found;
  outer:
  for (final row in matrix) {
    for (final val in row) {
      if (val > 5) {
        found = val;
        break outer;
      }
    }
  }
  print('First > 5: $found'); // 6

  for (var i = 1; i <= 10; i++) {
    if (i.isEven) continue;
    print(i);
  }
}
```

Метка (`outer:`) ставится перед циклом и позволяет `break` / `continue` адресовать не ближайший, а именно тот цикл. `break outer` завершает внешний `for`. `continue` пропускает оставшееся тело текущей итерации и переходит к следующей.
