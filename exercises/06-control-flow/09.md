<summary>Решение</summary>

```dart
int safeDivide(int a, int b) {
  try {
    if (b == 0) throw ArgumentError('b must not be zero');
    return a ~/ b;
  } on ArgumentError catch (e) {
    print('Caught: ${e.message}');
    return 0;
  } on Exception catch (e) {
    print('Unexpected: $e');
    rethrow;
  } finally {
    print('Always runs');
  }
}

void main() {
  print(safeDivide(10, 2)); // Always runs \n 5
  print(safeDivide(10, 0)); // Caught: b must not be zero \n Always runs \n 0
}
```

`try` / `on ExceptionType catch (e)` — перехват конкретного типа. Несколько `on`-блоков проверяются по порядку. `rethrow` повторно бросает **то же** исключение, сохраняя стек вызовов. `finally` выполняется **всегда** — независимо от того, было ли исключение.
