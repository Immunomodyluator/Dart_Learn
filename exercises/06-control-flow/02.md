# Задание 02 (6.1): if / else / switch — 🐛 Найти баг

**Тема:** 06. Поток управления | **Подтема:** 6.1 if / else / switch

---

Функция всегда возвращает `'unknown'`, несмотря на корректный ввод. Найди и исправь баги.

```dart
String dayType(String day) {
  switch (day) {
    case 'Monday':
    case 'Tuesday':
    case 'Wednesday':
    case 'Thursday':
    case 'Friday':
      print('Weekday');     // забыли возврат
    case 'Saturday':
    case 'Sunday'           // нет двоеточия
      return 'Weekend';
  }
  return 'unknown';
}

void main() {
  print(dayType('Monday'));   // ожидается Weekday
  print(dayType('Saturday')); // ожидается Weekend
}
```

<details>
<summary>Решение</summary>

```dart
String dayType(String day) {
  switch (day) {
    case 'Monday':
    case 'Tuesday':
    case 'Wednesday':
    case 'Thursday':
    case 'Friday':
      return 'Weekday';    // нужен return, а не print
    case 'Saturday':
    case 'Sunday':         // добавлено двоеточие
      return 'Weekend';
  }
  return 'unknown';
}

void main() {
  print(dayType('Monday'));   // Weekday
  print(dayType('Saturday')); // Weekend
}
```

**Два бага:** 1) `print('Weekday')` — выводит на экран, но не возвращает значение; функция проваливается в `return 'unknown'`. Нужен `return 'Weekday'`. 2) `case 'Sunday'` без `:` — синтаксическая ошибка компиляции.

</details>
