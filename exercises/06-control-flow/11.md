# Задание 11 (6.6): assert — 🖊 Написать код

**Тема:** 06. Поток управления | **Подтема:** 6.6 assert

---

Напиши функцию `createUser({required String name, required int age})`, которая:

1. Использует `assert` для проверки инварiantов: имя не пустое, возраст 0–150.
2. Возвращает `Map<String, Object>` с полями `name` и `age`.

Также напиши функцию `testAsserts()`, которая демонстрирует, что `assert` сработает в debug-режиме при нарушении условия.

> Замечание: `assert` активен только в **debug**-режиме (`dart run`). При компиляции с `dart compile exe` — игнорируется.

<details>
<summary>Решение</summary>

```dart
Map<String, Object> createUser({
  required String name,
  required int age,
}) {
  assert(name.isNotEmpty, 'Name must not be empty');
  assert(age >= 0 && age <= 150, 'Age must be between 0 and 150, got $age');
  return {'name': name, 'age': age};
}

void testAsserts() {
  // Сработает AssertionError в debug-режиме:
  // createUser(name: '', age: 25);
  // createUser(name: 'Alice', age: -1);

  final user = createUser(name: 'Alice', age: 30);
  print(user); // {name: Alice, age: 30}
}

void main() => testAsserts();
```

`assert(condition, message)` — бросает `AssertionError` с сообщением, если `condition` ложно. Используй `assert` для документирования предусловий и обнаружения ошибок разработчика во время тестирования. Для продакшн-валидации пользовательского ввода используй `throw ArgumentError`.
