<summary>Решение</summary>

```dart
String classify(int code) {
  if (code >= 200 && code <= 299) {
    return 'Success';
  } else if (code >= 400 && code <= 499) {
    return 'Client error';
  } else if (code >= 500) {
    return 'Server error';
  } else {
    return 'Redirect or info';
  }
}

void main() {
  print(classify(200)); // Success
  print(classify(404)); // Client error
  print(classify(500)); // Server error
  print(classify(301)); // Redirect or info
}
```

`if` / `else if` / `else` — классическая цепочка условий. Порядок веток важен: условия проверяются сверху вниз, выполняется первая истинная ветка. В Dart 3 этот же код можно переписать через `switch`-выражение с guard-условием (`when`).
