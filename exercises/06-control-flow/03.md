# Задание 03 (6.2): Циклы for / while / do-while — 🖊 Написать код

**Тема:** 06. Поток управления | **Подтема:** 6.2 Циклы for / while / do-while

---

Напиши три версии функции `countdown(int from)`, которая печатает числа от `from` до 1 включительно, каждое на новой строке, и затем `'Go!'`.

1. `countdownFor` — используй `for` с инициализатором.
2. `countdownWhile` — используй `while`.
3. `countdownDoWhile` — используй `do-while`.

Для `from <= 0` ничего не печатать, `'Go!'` тоже.

<details>
<summary>Решение</summary>

```dart
void countdownFor(int from) {
  for (var i = from; i >= 1; i--) {
    print(i);
  }
  if (from >= 1) print('Go!');
}

void countdownWhile(int from) {
  var i = from;
  while (i >= 1) {
    print(i);
    i--;
  }
  if (from >= 1) print('Go!');
}

void countdownDoWhile(int from) {
  if (from < 1) return;
  var i = from;
  do {
    print(i);
    i--;
  } while (i >= 1);
  print('Go!');
}

void main() {
  countdownFor(3);
  // 3  2  1  Go!
}
```

`for (init; cond; step)` — компактно когда шаг известен заранее. `while` — когда условие проверяется до входа. `do-while` — тело выполняется **хотя бы раз**; поэтому нужна отдельная проверка `from < 1` перед входом в `do-while`.
