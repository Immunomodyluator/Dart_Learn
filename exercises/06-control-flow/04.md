# Задание 04 (6.2): Циклы for / while / do-while — 🐛 Найти баг

**Тема:** 06. Поток управления | **Подтема:** 6.2 Циклы for / while / do-while

---

Функция должна найти первый чётный элемент и вернуть его. Код зависает или возвращает неверный результат. Исправь все ошибки.

```dart
int? firstEven(List<int> nums) {
  var i = 0;
  while (i < nums.length) {
    if (nums[i].isEven) {
      return nums[i];
    }
    // забыли инкремент
  }

  for (var j = 0; j < nums.length; j++) {
    if (j.isEven) return nums[j];  // проверяем индекс, а не элемент
  }

  return null;
}

void main() {
  print(firstEven([1, 3, 4, 6])); // ожидается 4
  print(firstEven([1, 3, 5]));    // ожидается null
}
```

<details>
<summary>Решение</summary>

```dart
int? firstEven(List<int> nums) {
  var i = 0;
  while (i < nums.length) {
    if (nums[i].isEven) {
      return nums[i];
    }
    i++;          // инкремент — иначе бесконечный цикл
  }
  return null;
}

void main() {
  print(firstEven([1, 3, 4, 6])); // 4
  print(firstEven([1, 3, 5]));    // null
}
```

**Два бага:** 1) В `while`-цикле нет `i++` — при нечётном первом элементе цикл бесконечен. 2) Второй `for`-цикл проверяет `j.isEven` (чётность **индекса**), а не `nums[j].isEven` (чётность **элемента**). Второй цикл лишний — первый уже решает задачу.
