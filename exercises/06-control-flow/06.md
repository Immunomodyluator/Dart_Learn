# Задание 06 (6.3): break / continue / labels — 🐛 Найти баг

**Тема:** 06. Поток управления | **Подтема:** 6.3 break / continue / labels

---

Код должен напечатать сумму элементов до первого отрицательного числа. Вместо этого он суммирует все элементы. Найди и исправь ошибку.

```dart
void main() {
  final data = [3, 7, 2, -1, 5, 8];
  var sum = 0;

  for (final n in data) {
    if (n < 0) continue;  // хотели остановиться, но пропустили элемент
    sum += n;
  }

  print('Sum before negative: $sum'); // ожидается 12
}
```

<details>
<summary>Решение</summary>

```dart
void main() {
  final data = [3, 7, 2, -1, 5, 8];
  var sum = 0;

  for (final n in data) {
    if (n < 0) break;   // break — полностью остановить цикл
    sum += n;
  }

  print('Sum before negative: $sum'); // 12  (3+7+2)
}
```

`continue` **пропускает** текущую итерацию и переходит к следующей — цикл продолжается. Чтобы **остановить** цикл при встрече отрицательного числа, нужен `break`. В данном случае `continue` заставлял код суммировать все ненегативные элементы, включая `5` и `8`.
