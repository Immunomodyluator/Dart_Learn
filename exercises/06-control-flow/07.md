# Задание 07 (6.4): Паттерн-матчинг switch — 🖊 Написать код

**Тема:** 06. Поток управления | **Подтема:** 6.4 Паттерн-матчинг switch (Dart 3)

---

Напиши функцию `describeShape(Object shape)`, которая использует Dart 3 `switch`-выражение с паттернами для возврата строки-описания.

Поддерживаемые входные значения:

- `(double r)` — запись с одним double → `'Circle with radius $r'`
- `(double w, double h)` — запись с двумя double → `'Rectangle ${w}x${h}'`
- `int n` где `n > 0` → `'Regular polygon with $n sides'`
- всё остальное → `'Unknown shape'`

<details>
<summary>Решение</summary>

```dart
String describeShape(Object shape) {
  return switch (shape) {
    (double r,) => 'Circle with radius $r',               // запись 1-поле
    (double w, double h) => 'Rectangle ${w}x${h}',        // запись 2-поля
    int n when n > 0 => 'Regular polygon with $n sides',  // guard
    _ => 'Unknown shape',
  };
}

void main() {
  print(describeShape((5.0,)));           // Circle with radius 5.0
  print(describeShape((3.0, 4.0)));       // Rectangle 3.0x4.0
  print(describeShape(6));               // Regular polygon with 6 sides
  print(describeShape('triangle'));      // Unknown shape
}
```

Dart 3 `switch`-выражение проверяет каждый `case`-паттерн сверху вниз. `(double r,)` — паттерн однопольного Record. `when` добавляет guard-условие поверх структурного соответствия. `_` — wildcard, совпадает с чем угодно.
