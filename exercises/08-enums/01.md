Дополни пропуски: объяви enum и используй его в switch-выражении.

```dart
// Объявить enum с вариантами: pending, active, suspended, deleted
___ UserStatus { ___, ___, ___, ___ }

String statusLabel(UserStatus s) => switch (s) {
  UserStatus.___ => 'Ожидает',
  UserStatus.___ => 'Активен',
  UserStatus.___ => 'Заблокирован',
  UserStatus.___ => 'Удалён',
};

void main() {
  for (final s in UserStatus.___) {   // итерация по всем значениям
    print('${s.___}: ${statusLabel(s)}');  // имя значения
  }
}
```

<details>
<summary>Решение</summary>

```dart
enum UserStatus { pending, active, suspended, deleted }

String statusLabel(UserStatus s) => switch (s) {
  UserStatus.pending   => 'Ожидает',
  UserStatus.active    => 'Активен',
  UserStatus.suspended => 'Заблокирован',
  UserStatus.deleted   => 'Удалён',
};

void main() {
  for (final s in UserStatus.values) {
    print('${s.name}: ${statusLabel(s)}');
  }
}
// pending: Ожидает
// active: Активен
// suspended: Заблокирован
// deleted: Удалён
```

`enum Foo { a, b, c }` — объявление. `Foo.values` — список всех вариантов в порядке объявления. `value.name` — строковое имя варианта (`'pending'` и т.д.). `switch` по `sealed enum` должен быть исчерпывающим — компилятор это проверяет.

</details>
