# Задание 03 (8.2): Расширенные enum (поля, методы, интерфейсы) — 🖊 Написать код

**Тема:** 08. Перечисления | **Подтема:** 8.2 Расширенные enum

---

Объяви расширенный enum `Planet` с полями и методами (Dart 2.17+):

- Поля: `double massKg` и `double radiusM` (константные).
- Варианты: `mercury`, `venus`, `earth`, `mars` — с реальными (приблизительными) значениями.
- Метод `double surfaceGravity()` → `6.674e-11 * massKg / (radiusM * radiusM)`.
- Метод `double surfaceWeight(double bodyMass)` → `bodyMass * surfaceGravity()`.

Пример:

```
// Вес 75 кг на каждой планете
for (final p in Planet.values) {
  print('${p.name}: ${p.surfaceWeight(75).toStringAsFixed(2)} N');
}
```

<details>
<summary>Решение</summary>

```dart
enum Planet {
  mercury(3.303e+23, 2.4397e6),
  venus  (4.869e+24, 6.0518e6),
  earth  (5.976e+24, 6.37814e6),
  mars   (6.421e+23, 3.3972e6);

  const Planet(this.massKg, this.radiusM);

  final double massKg;
  final double radiusM;

  static const double _g = 6.674e-11;

  double surfaceGravity() => _g * massKg / (radiusM * radiusM);
  double surfaceWeight(double bodyMass) => bodyMass * surfaceGravity();
}

void main() {
  for (final p in Planet.values) {
    print('${p.name}: ${p.surfaceWeight(75).toStringAsFixed(2)} N');
  }
  // mercury: 28.33 N
  // venus:   67.89 N
  // earth:   73.51 N
  // mars:    27.88 N
}
```

Расширенный `enum` позволяет объявлять поля, конструктор `const`, методы и реализовывать интерфейсы. Каждый вариант передаёт аргументы в `const`-конструктор. Поля должны быть `final`. Точка с запятой `;` после последнего варианта обязательна, если есть тело enum.
