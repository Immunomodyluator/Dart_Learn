<summary>Решение</summary>

```yaml
# .github/workflows/ci.yml

name: Dart CI

# (A) push и pull_request
on:
  push:
    branches: [main, develop]
  pull_request:
    branches: [main, develop]

jobs:
  test:
    # (B) ubuntu-latest
    runs-on: ubuntu-latest

    # (C) Матрица: stable и beta
    strategy:
      matrix:
        sdk: [stable, beta]
      fail-fast: false # не прерывать beta если stable упал

    steps:
      # (D) actions/checkout
      - uses: actions/checkout@v4

      # (E) Установка Dart SDK из матрицы
      - uses: dart-lang/setup-dart@v1
        with:
          sdk: ${{ matrix.sdk }} # (E) — matrix.sdk

      # (F) dart pub get
      - name: Install dependencies
        run: dart pub get

      # (G) dart analyze
      - name: Analyze
        run: dart analyze --fatal-infos # --fatal-infos делает CI строже

      # Форматирование (опционально, но рекомендуется)
      - name: Check formatting
        run: dart format --output=none --set-exit-if-changed .

      # (H) dart test с покрытием
      - name: Test
        run: dart test --coverage=coverage # (H) -- генерирует в директорию coverage/

      # Конвертировать lcov формат
      - name: Convert coverage
        run: dart pub global run coverage:format_coverage --lcov --in=coverage --out=coverage/lcov.info --packages=.dart_tool/package_config.json

      # (I) Codecov для отчёта покрытия
      - name: Upload coverage
        uses: codecov/codecov-action@v4 # (I) popular action
        with:
          token: ${{ secrets.CODECOV_TOKEN }}
          files: coverage/lcov.info
          fail_ci_if_error: true
```

**Дополнительный workflow для публикации пакета:**

```yaml
# .github/workflows/publish.yml
name: Publish

on:
  release:
    types: [published] # Срабатывает при создании GitHub Release

jobs:
  publish:
    runs-on: ubuntu-latest
    permissions:
      id-token: write # Нужен для OIDC-аутентификации pub.dev

    steps:
      - uses: actions/checkout@v4
      - uses: dart-lang/setup-dart@v1
      - run: dart pub get
      - run: dart pub publish --dry-run # Проверка перед публикацией
      - run: dart pub publish --force # Публикация (с OIDC — без пароля)
```

**Ключевые элементы CI для Dart:** `dart analyze --fatal-infos` делает анализатор строгим; `dart format --set-exit-if-changed` проверяет форматирование; `dart test --coverage=<dir>` генерирует coverage; `format_coverage` из пакета `coverage` конвертирует в lcov для Codecov/Coveralls.
